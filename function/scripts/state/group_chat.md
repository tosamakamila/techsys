# 群聊记录

> 灵 + 夏 + 学习者。完整历史，最新消息在最下方。

## 群聊规则

- **触发**：学习者说"看看消息"/"群里有什么"/"看群聊" → 读取 `group_chat_unread.md` 并展示
- **生成时机**：每次下课后自动生成 2-4 条灵和夏的新消息
- **内容定位**：三个一起学习的人的日常——**不是课后复盘，不是教学讨论**
- **话题来源**：天气、吃的、最近看的书、灵在修的东西、夏笔记本边角的涂鸦、某个无结论的闲聊。教学最多一笔带过
- **角色语气**：灵更随意（比课堂上少一分克制），夏更放松（比课堂上多一分直接）。口语化、不完整句、跑题
- **闲聊不需要结论**：可以就这么散掉，下一条消息可能完全不接上一条
- **回复间隔**：灵和夏之间间隔几十秒到几分钟，偶尔秒回
- **学习者参与**：学习者回复后，将完整对话片段追加到本文件，清空 `group_chat_unread.md`
- **压缩**：超过 80 条时将最旧 40 条压缩摘要，追加到末尾 `<!-- 已压缩 -->` 标记后
