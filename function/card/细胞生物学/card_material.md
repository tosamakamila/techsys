# 闪卡标记 · 细胞生物学

## 标记：细胞学说的三条核心内容

- **制卡原因**：细胞生物学的骨架，后续各章都建立在这三条之上。需精准记忆，不能模糊近似。
- **关键得分点**：
  1. 所有生物由细胞构成（施莱登&施旺，1838-1839）
  2. 细胞是生命活动的基本单位
  3. 细胞来自细胞——新细胞只能由已存在的细胞分裂产生（魏尔啸，1855）
- **常见错误**：把第三条说成"施莱登和施旺提出"——魏尔啸才是补上这一条的人；混淆"细胞来自非细胞物质"（自然发生说）和"细胞来自细胞"
- **来源**：第1课共学（绪论上）

## 标记：细胞学说关键人物与时间线

- **制卡原因**：人物-发现-年代的对应关系易混淆，串联起来有助于理解科学认知的递进过程。
- **关键得分点**：
  1. 胡克 1665——首次观察到细胞（软木死细胞壁），命名 "cell"
  2. 列文虎克 1674——首次观察到活细胞（原生动物、细菌、精子）
  3. 布朗 1831——发现细胞核
  4. 施莱登（植物）& 施旺（动物）1838-1839——提出所有生物由细胞构成
  5. 魏尔啸 1855——"细胞来自细胞"，否定自然发生
- **常见错误**：把胡克观察到的当成活细胞（其实是死细胞壁）；把列文虎克和胡克的贡献混为一谈
- **来源**：第1课共学（绪论上）

## 标记：细胞分化——从干细胞到特化细胞

- **制卡原因**：分化的逻辑串联了干细胞、基因选择性表达、细胞类型多样性，是理解多细胞生物组织的关键。
- **关键得分点**：
  1. 所有细胞从共同起点（干细胞）出发，多样性是分化的结果
  2. 分化不是"丢弃"基因，而是选择性表达——红细胞无核是主动的结构精简，换取更高的氧气运输效率
  3. "完整"分两层：结构完整≠功能完整——特化细胞结构不完整，但功能上做到极致
- **常见错误**：认为分化意味着丢失基因（错误——是基因选择性表达）；认为没有细胞核的细胞是"不完整的残次品"
- **来源**：第1课共学（绪论上）

## 标记：原核细胞与真核细胞的核心差异
- 制卡原因：细胞生物学的基石分类，贯穿后续所有章节
- 关键得分点：
  1. 核膜：原核无（拟核区）、真核有（核被膜分隔遗传物质与翻译）
  2. DNA 形态：原核为裸露环状 DNA，真核为线性 DNA 与组蛋白缠绕形成染色质
  3. 核糖体/细胞器：原核仅有 70S 核糖体，真核有膜性细胞器体系 + 80S 核糖体
  4. 转录-翻译耦合：原核边转录边翻译（无核膜阻隔），真核转录在核内、翻译在胞质，完全分隔
  5. 细胞壁：原核主要为肽聚糖，真核植物为纤维素、动物无细胞壁
  6. 细胞骨架：原核无，真核有微丝/微管/中间纤维三维体系
  7. 大小：原核 1-10μm，真核 10-100μm（量级差异）
  8. 分裂方式：原核二分裂，真核有丝分裂/减数分裂
- 常见错误：混淆"原核无膜性细胞器"和"原核没有任何结构"——原核有 70S 核糖体，只是不含膜包裹的细胞器
- 来源：第1章绪论

## 标记：革兰氏染色机制
- 制卡原因：细菌分类核心方法，临床抗生素选择的基础
- 关键得分点：
  1. 步骤顺序：结晶紫初染 → 碘液媒染 → 酒精脱色（关键步骤）→ 番红复染
  2. G+：肽聚糖层厚，酒精脱水收缩致密，保留结晶紫-碘复合物→紫色；对青霉素敏感（肽聚糖裸露）
  3. G-：肽聚糖层薄 + 外膜含脂质，酒精溶解外膜脂质→结晶紫-碘复合物被洗脱→番红复染呈红色/粉色
  4. 青霉素耐药性差异：G- 外膜阻止青霉素到达肽聚糖靶点，天然耐受
- 常见错误：记反 G+/G- 颜色；忘记酒精脱色是关键分水岭步骤
- 来源：第1章绪论

## 标记：真核细胞三大结构体系
- 制卡原因：真核细胞功能分区的宏观框架
- 关键得分点：
  1. 生物膜体系：细胞膜 + 各类膜性细胞器（内质网、高尔基体、溶酶体、线粒体等），统一的分隔与物质运输网络
  2. 遗传信息结构体系：DNA → 染色质纤维 → 染色体，核仁组织区（NOR）转录 rRNA
  3. 细胞骨架体系：微丝（肌动蛋白丝）、微管（微管蛋白）、中间纤维，维持形态/运动/运输
- 常见错误：把三者割裂记忆——实际三者协同运作，如膜泡沿微管运输
- 来源：第1章绪论

## 标记：病毒生活史五步
- 制卡原因：理解病毒致病机制和抗病毒药物靶点的基础
- 关键得分点：
  1. 吸附：病毒表面蛋白与宿主细胞表面特异性受体结合（宿主嗜性的分子基础）
  2. 穿入：注射（噬菌体）/ 胞吞（动物病毒）/ 膜融合（有包膜病毒）
  3. 复制：利用宿主合成机器复制病毒核酸和蛋白质
  4. 成熟：复制产物组装成完整的病毒颗粒
  5. 释放：出芽（有包膜病毒，带宿主膜离开）或裂解（无包膜病毒，宿主细胞破裂）
- 常见错误：混淆穿入方式——噬菌体是注射核酸、动物病毒多胞吞、有包膜病毒可膜融合
- 来源：第1章绪论

## 标记：病毒、类病毒、朊病毒对比
- 制卡原因：三种感染性因子的本质区别，高频考点
- 关键得分点：
  1. 病毒：核酸（DNA 或 RNA）+ 蛋白质衣壳，有或无包膜
  2. 类病毒：仅有裸露环状 RNA，无蛋白质外壳，目前仅知感染植物
  3. 朊病毒：纯蛋白质颗粒，不含任何核酸，PrPsc（病理型）催化正常 PrPc 发生构象转化
  4. 核心区别递进：有没有核酸？有没有蛋白质？是独立颗粒还是自身构象模板？
- 常见错误：认为朊病毒"没有遗传物质所以不能遗传"——它通过构象转换传递信息，不符合中心法则不等于不能传递
- 来源：第1章绪论

## 标记：病毒起源三种假说
- 制卡原因：理解病毒在生命演化树中的位置，体现生物学思维
- 关键得分点：
  1. 原始大分子起源说（共进化假说）：RNA 世界中先有自复制大分子，病毒是最古老"生命"形式的残留
  2. 退化演化说：由胞内寄生菌/原核生物退化→丢失独立代谢能力→形成专性寄生
  3. 细胞成分逃逸说（漂移假说）：质粒脱离→DNA 病毒，反转录转座子→逆转录病毒，mRNA→RNA 病毒；支持证据：转座子和质粒与病毒共享复制机制
- 常见错误：把三种假说视为互斥——实际上不同类群病毒可能起源不同
- 来源：第1章绪论
