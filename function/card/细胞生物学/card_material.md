# 闪卡标记 · 细胞生物学

## 标记：细胞学说的三条核心内容
- **考频/出题**：必考。简答"简述细胞学说主要内容"或选择判断
- **得分锚点**：
  1. 构成：所有生物由细胞构成——施莱登&施旺，1838-1839
  2. 单位：细胞是生命活动的基本单位——同上
  3. 来源：细胞来自细胞——**魏尔啸**，1855（这人是后来补刀的，不跟施莱登施旺绑定）
- **易错陷阱**：第三条错配给施莱登施旺，这是出题人最爱设的坑。三条递进关系（构成→单位→来源）理清了就不会搞错人
- **记忆钩子**：施莱登=植物、施旺=动物，两人合伙搞定前两条。魏尔啸单独补第三条——"后来的那个人"
- **来源**：第1课（绪论上）

## 标记：细胞学说关键人物与时间线
- **考频/出题**：常考选择/填空题，尤其人物-发现-年份的匹配
- **得分锚点**：按时间轴排列——
  1. 胡克 1665：首次见细胞（死细胞壁），命名"cell"
  2. 列文虎克 1674：首次见**活**细胞
  3. 布朗 1831：细胞核
  4. 施莱登&施旺 1838-1839：细胞学说前两条
  5. 魏尔啸 1855：细胞来自细胞
- **易错陷阱**：胡克vs列文虎克——一个看死的，一个看活的。胡克命名cell但不是第一个看活细胞的人。题干写"首次观察到活细胞"→列文虎克，不是胡克
- **记忆钩子**：胡克=hook=勾住了cell这个名字。列文虎克="活"字在前。时间轴每行一个世纪（17→18→19→19→19）
- **来源**：第1课（绪论上）

## 标记：细胞分化——基因选择性表达
- **考频/出题**：常考概念辨析。"分化是否丢失基因"是经典判断题
- **得分锚点**：
  1. 起点：干细胞（全能），终点：特化细胞（极致专一）
  2. 机制：**不是丢基因，是选择性表达**——有些基因关掉了
  3. 案例：红细胞无核=主动精简结构=换氧气运输效率。不是"不完整"，是"特化到极致"
- **易错陷阱**：学生总把"选择性表达"说成"丢失基因"——换到考试答题里直接零分。另外"无核=残次品"也是送命题。无核细胞=主动放弃细胞核换功能=分化策略，不是退化
- **记忆钩子**：分化=基因开关灯，不是删掉基因。红细胞无核=赛车拆掉后排座位——不是事故，是改装
- **来源**：第1课（绪论上）

## 标记：原核与真核八大维度对比
- **考频/出题**：高频。表格填空、判断、多选题都出过
- **得分锚点**（逐维对比，按这八条答）：
  1. 核膜：原核无（拟核），真核有
  2. DNA：原核环状裸露，真核线性+组蛋白→染色质
  3. 核糖体：原核 70S，真核 80S（+膜性细胞器体系）
  4. 转录-翻译：原核**边转录边翻译**（无核膜阻隔），真核分开
  5. 细胞壁：原核=肽聚糖，真核植物=纤维素，真核动物=无
  6. 细胞骨架：原核无，真核有（微丝/微管/中间纤维）
  7. 大小：原核 1-10μm，真核 10-100μm
  8. 分裂：原核二分裂，真核有丝/减数
- **易错陷阱**："原核无膜性细胞器"≠"原核什么都没"——它有 70S 核糖体。转录-翻译耦合是原核特征，别反记了
- **记忆钩子**：真核=有膜包核+每个结构都更复杂。原核=精简版，但转录翻译耦合是它的"独门绝活"
- **来源**：第1章绪论

## 标记：革兰氏染色——步骤、机制、临床意义
- **考频/出题**：选择/填空高频。尤其染色步骤顺序和G+/G-颜色区分
- **得分锚点**：
  1. 四步顺序：结晶紫初染→碘液媒染→**酒精脱色（分水岭）**→番红复染
  2. G+（阳性=紫）：肽聚糖**厚**，酒精脱水收缩→锁住结晶紫-碘→紫色
  3. G-（阴性=红/粉）：肽聚糖**薄**+外膜含脂→酒精溶掉外膜→染料洗脱→番红复染
  4. 青霉素：对G+有效（肽聚糖裸露），对G-无效（外膜挡住）
- **易错陷阱**：记反G+/G-颜色。记法：G+是"加号的帝国"→城墙厚→紫（帝国紫）。酒精脱色是区分性步骤，别跳过
- **记忆钩子**：G+厚壁=紫（帝国紫），G-薄壁+外膜=红。紫色染料→壁厚的留住→正号对应正紫色
- **来源**：第1章绪论

## 标记：真核细胞三大结构体系
- **考频/出题**：论述/简答。问"真核细胞的结构体系"而非"有哪些细胞器"
- **得分锚点**：
  1. 生物膜体系：细胞膜+膜性细胞器→统一的分隔与运输网络
  2. 遗传信息体系：DNA→染色质纤维→染色体，NOR转录rRNA
  3. 细胞骨架体系：微丝+微管+中间纤维→形态/运动/运输
- **易错陷阱**：把三者当独立体系——考试爱问"三者如何协同"。膜泡沿微管运输=膜体系+骨架体系协同。不是三个孤岛
- **记忆钩子**：膜=墙和房间，遗传=图纸，骨架=钢架。一栋楼三个系统
- **来源**：第1章绪论

## 标记：病毒生活史五步
- **考频/出题**：简答"简述病毒增殖过程"或选择题考某一步的细节
- **得分锚点**：
  1. 吸附：病毒蛋白↔宿主受体，决定了"谁能感染谁"
  2. 穿入：噬菌体=注射核酸（衣壳留在外面），动物病毒=胞吞或膜融合
  3. 复制（广义）：利用宿主机器复制核酸+合成蛋白
  4. 成熟：组装成完整病毒颗粒
  5. 释放：有包膜=出芽（带宿主膜走），无包膜=裂解（宿主破裂）
- **易错陷阱**：穿入方式最容易混——噬菌体是打针（注射核酸），动物病毒是吞进去（胞吞）或膜融合。别把噬菌体的注射穿入套到动物病毒上
- **记忆钩子**：噬菌体像个注射器，扎进去核酸就完事，壳不要了。动物病毒比较"讲究"，整个进去
- **来源**：第1章绪论

## 标记：病毒、类病毒、朊病毒——三种感染因子
- **考频/出题**：选择/判断高频。"下列关于朊病毒的说法正确的是"是经典坑题
- **得分锚点**（递进对比）：
  1. 病毒：核酸+衣壳，有或无包膜
  2. 类病毒：**裸RNA**，无蛋白外壳，仅知感染植物
  3. 朊病毒：**纯蛋白**，无核酸，PrPsc催化PrPc构象转化
- **易错陷阱**：朊病毒"无遗传物质"≠"不能传递信息"——它通过构象转换传信息，不走中心法则。类病毒vs病毒=有无衣壳。朊病毒vs类病毒=有无核酸
- **记忆钩子**：病毒=完整包，类病毒=裸RNA，朊病毒=光杆蛋白。越来越"简陋"，但都致病
- **来源**：第1章绪论

## 标记：病毒起源三种假说
- **考频/出题**：简答"简述病毒起源假说"。三种不是互斥的——不同类群可能不同来源
- **得分锚点**：
  1. 原始大分子起源（共进化）：RNA世界中先有自复制大分子→病毒是古老生命残留
  2. 退化演化：胞内寄生菌退化→丢失独立代谢→专性寄生
  3. 细胞成分逃逸（漂移）：质粒→DNA病毒，反转座子→逆转录病毒，mRNA→RNA病毒
- **易错陷阱**：三种假说不是三选一。考试特别喜欢考"三种假说互斥吗？"→答案是"不一定，不同类群可能起源不同。"漂移假说的证据最强（转座子/质粒与病毒共享复制机制）
- **记忆钩子**：一个"本来就有的"（共进化），一个"退下来的"（退化），一个"跑出来的"（逃逸）
- **来源**：第1章绪论

## 标记：细胞、原生质、原生质体——概念辨析
- **考频/出题**：选择/判断题常考。"下列关于原生质的描述正确的是"易踩坑
- **得分锚点**：
  1. 细胞（结构单位）：膜包围、能独立繁殖的最小原生质团
  2. 原生质（物质概念）：细胞内所有生活物质的总称——是**化学成分**，不是结构
  3. 原生质体（结构单位）：脱去细胞壁后的完整细胞结构——球形
- **易错陷阱**：原生质=物质 vs 原生质体=结构。考卷最爱把两者互换。细胞≠原生质体（前者有壁/无壁均可，后者特指脱壁后的球状体）
- **记忆钩子**：原生质=质，看到"质"=物质=成分。原生质体=体，看到"体"=结构=一个球
- **来源**：第1章绪论第4节

## 标记：酶六大分类——分类标准是什么
- **考频/出题**：选择题"不属于水解酶的是"或填空题"按反应类型分为哪六类"
- **得分锚点**（按反应类型分，不是按结构分）：
  1. 氧化还原酶：电子/氢原子转移
  2. 转移酶：基团在分子间转移
  3. 水解酶：加水分解（蛋白酶、脂肪酶）
  4. 裂解酶：非水解断键→生成双键或加至双键
  5. 异构酶：同分异构互变
  6. 连接酶（合成酶）：两分子连接，需ATP
- **易错陷阱**：分类标准是**反应类型**不是结构。水解酶（加水）和裂解酶（不加水直接断）的区别是送命题——题干写"不加水断键"→裂解酶不是水解酶
- **记忆钩子**：氧转电，转基团，水解加水，裂解脱碳，异构翻转，连接消耗ATP
- **来源**：第1章绪论第6节

## 标记：代谢网络——呼吸作用是代谢中枢
- **考频/出题**：简答"为什么呼吸作用被称为代谢中枢"
- **得分锚点**：
  1. 代谢通路通过共享中间产物串联成**网络**，不是独立管道
  2. 呼吸作用同时产出ATP（能量货币）和碳骨架（合成前体）
  3. 正因如此它连接几乎所有代谢通路→称为"代谢中枢"
- **易错陷阱**：别把代谢通路想象成一条条独立的线——中间产物交叉流动。答题只写"产生ATP"不完整，碳骨架供合成同样重要
- **记忆钩子**：呼吸作用=火车站，ATP=电力，碳骨架=铁轨——其他通路都要经过这个站
- **来源**：第1章绪论第6节

## 标记：细菌鞭毛——顶部延伸
- **考频/出题**：选择题/判断题常考。"鞭毛的生长方式"是反直觉考点
- **得分锚点**：
  1. 组成：鞭毛蛋白亚基螺旋排列
  2. 生长方式：**顶部延伸**——新亚基从顶端加入（不是从基部延长）
  3. 机制：鞭毛蛋白通过中空孔道运输到顶端组装
  4. 基部：鞭毛钩连接马达，旋转驱动
- **易错陷阱**：正常逻辑"从根部长"→但细菌鞭毛是从尖部长。题干"从基部延伸"一定错
- **记忆钩子**：鞭毛是一根空心管，蛋白质从洞里运到最远端加上去——像挤牙膏从出口处堆起来
- **来源**：第1章绪论第8节

## 标记：质粒——核外环状DNA
- **考频/出题**：选择/填空。质粒≠必需遗传物质——是附加装备
- **得分锚点**：
  1. 本质：核外环状DNA，独立于拟核复制
  2. 性质：非必需，丢了也活——带的是"锦上添花"的附加性状
  3. 三大代表：F因子（接合/性因子）、Col因子（大肠杆菌素→杀竞争者）、R因子（耐药性→菌群间传播）
- **易错陷阱**：以为质粒是细菌必需的——不是。去质粒的细菌照样活。质粒=背包里的附加装备。R因子耐药性可传播→临床应用上的大麻烦
- **记忆钩子**：质=附加品质，粒=小粒DNA。F=性（繁殖优势），Col=杀（灭竞争者），R=抗（耐药防御）
- **来源**：第1章绪论第8节

## 标记：芽孢——休眠体不是繁殖方式
- **考频/出题**：判断/选择高频。"芽孢是真菌的繁殖方式吗？"——不是，是细菌的休眠体
- **得分锚点**：
  1. 本质：G+细菌在不利环境下的**休眠体**（内生孢子）
  2. 抗逆性：耐热、耐干燥、耐辐射、耐化学消毒——代谢几乎停止
  3. 繁殖方式：细菌只有**二分裂**。一个细菌=一个芽孢→不增加个体数
- **易错陷阱**：芽孢=休眠≠繁殖。真菌孢子才是繁殖。考试设坑：把芽孢写成"繁殖结构"——直接判错。"一个细菌产生多个芽孢"——也错
- **记忆钩子**：芽孢=细菌的"冬眠帐篷"——一个细菌搭一个帐篷，不会变两个
- **来源**：第1章绪论第8节

## 标记：蓝细菌——异形胞与蓝色体
- **考频/出题**：选择/简答。异形胞的固氮功能是经典考点
- **得分锚点**：
  1. 能量：光能自养，质膜内陷形成光合片层=蓝色体（chromatophore）→含叶绿素a和藻胆素
  2. 异形胞（heterocyst）：缺氮时形成→细胞壁**加厚**→创造厌氧环境→保护固氮酶（对氧敏感）
  3. 分工：营养细胞光合产氧，异形胞固氮——两者互补但不兼容（固氮酶怕氧）
- **易错陷阱**：蓝细菌=原核生物，不是真核/藻类。异形胞和普通营养细胞功能不同——别混为一谈
- **记忆钩子**：异形胞=加了防氧罩的固氮车间。加厚细胞壁=把氧气挡在外面=固氮酶安全工作
- **来源**：第1章绪论第9节

## 标记：古细菌——与真核的进化关系
- **考频/出题**：简答"为什么说古细菌在进化上更接近真核"。洛基古菌是近年热点
- **得分锚点**：
  1. 形态上像原核（无核膜、环状DNA），但**基因和蛋白层面更接近真核**
  2. 细胞膜脂质：醚键连接（细菌是酯键）——基础结构就不同
  3. 洛基古菌（Lokiarchaeota）：拥有以前只在真核中发现的蛋白编码（肌动蛋白骨架相关）
  4. 意义：支持"两域系统"——真核起源于古细菌内部一个分支
- **易错陷阱**：古细菌≠细菌——名称有"细菌"但在基因层面更近真核。别把古细菌归入原核生物的传统分组
- **记忆钩子**：古细菌=穿了原核外套的真核近亲。洛基=Loki=北欧神话的"诡计之神"→它骗过了所有人的分类
- **来源**：第1章绪论第10节

## 标记：病毒分类四维度
- **考频/出题**：简答/选择。"按宿主/核酸/行为/巴尔的摩四种维度分类"
- **得分锚点**：
  1. 按宿主：噬菌体、植物病毒、动物病毒
  2. 按行为（与宿主关系）：温和（溶原、不急裂）、烈性（溶菌、立刻裂）
  3. 按核酸：DNA病毒、RNA病毒、蛋白质病毒（朊病毒）
  4. 巴尔的摩：按核酸种类及mRNA来源→七类
- **易错陷阱**：四维度互相独立。温和/烈性=行为分类，DNA/RNA=物质分类——不要交叉混淆。巴尔的摩分类是mRNA合成路径分类，不是简单按核酸
- **记忆钩子**：谁被感染（宿主）→怎么对待宿主（行为）→装了什么（核酸）→怎么表达mRNA（巴尔的摩）
- **来源**：第1章绪论第12节

## 标记：病毒侵入宿主的四种方式
- **考频/出题**：选择/填空。"噬菌体的侵入方式是"→注射。"有包膜病毒用"→膜融合
- **得分锚点**：
  1. 直接侵入：少数无包膜动物病毒，整个穿过细胞膜
  2. 注射：**噬菌体**——核酸注入宿主，衣壳留在外面
  3. 胞吞：动物病毒表面蛋白↔受体→触发胞吞→进囊泡
  4. 膜融合：**有包膜病毒**——包膜与宿主膜直接融合
- **易错陷阱**：噬菌体=注射（壳留在外面），不是胞吞。有包膜=膜融合，无包膜=胞吞或直接。侵入方式取决于病毒结构
- **记忆钩子**：噬菌体打针，有包膜抱上去融合，无包膜要么硬闯要么吞掉
- **来源**：第1章绪论第12节

## 标记：插入≠整合——病毒复制的两个层次
- **考频/出题**：判断/概念辨析。"所有病毒都会把核酸整合进宿主DNA"是高频错误
- **得分锚点**：
  1. 大多数病毒：核酸在胞质中直接利用宿主核糖体→转录/翻译→不碰宿主DNA（**不整合**）
  2. 整合是特例：逆转录病毒（HIV）→RNA逆转录为DNA→整合进宿主染色体→随细胞分裂复制
  3. "进入"≠"插入"——病毒核酸进入细胞≠整合进宿主DNA
- **易错陷阱**：把"病毒核酸进入细胞"当成"插入宿主DNA"——进入=到了细胞质，插入=整合进了DNA，两步差了一个数量级。逆转录病毒才做整合
- **记忆钩子**：整合=把病毒DNA写进了宿主染色体的剧本里。大多数病毒不这么做——它们在细胞质里搭临时棚拍完就走
- **来源**：第1章绪论第12节
