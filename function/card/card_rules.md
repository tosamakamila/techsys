# 闪卡制卡规则

子代理制作闪卡分两步：先写卡材（输入），再出卡片（输出）。两步不能省。

## 一、卡材怎么写（card_material.md）

卡材是老师的考试备忘，不是教材摘要。教材内容子代理自己能读到——卡材写的是**教材里没有的教学经验**。

### 四个字段

- **考频/出题**：这概念考试怎么出——简答/选择/判断/填空。一句话说清。
- **得分锚点**：阅卷人按什么给分。逐条列出，每条是独立得分单元。术语、数字、人名精准。编号，结构化。
- **易错陷阱**：往届学生实际掉过的坑。谁记反了、谁混淆了、谁写反了——具体到"把A当成B"才算有用。"需注意掌握"等于没写。
- **记忆钩子**：可用的类比或口诀。如果没有好的，不硬凑。

### 写作红线

- 不搬运教材原文。教材里有的不重复写。
- 不写"需精准记忆""要认真理解""该考点非常重要"——空话。
- 常见错误必须具体到"把X误认为Y"的程度，不能写"学生容易搞错"。
- 得分锚点用编号，每条 ≤ 一行。阅卷人是按条目给分的。
- 记忆钩子服务于快速回忆，不需要科学严谨。

## 二、卡片怎么写（cards.md）

从卡材的得分锚点和易错陷阱中，出真问题。

### 问题

- 必须是能触发主动回忆的**具体问题**，不是概念标题
  - "细胞学说的三条核心内容是什么？"
  - "简述革兰氏染色的步骤，哪一步是关键分水岭？"
  - "分化意味着丢失基因——这句话对吗？为什么？"
- 一张卡一个问，不要连环问
- 问题让学习者脑中必须先想，再翻面对照

### 答案

- 得分锚点结构化为编号列表
- 术语、数字百分百精准
- 只写对的，不解释为什么错（explanation 管这个）

### 解释

- 直接引用卡材的"易错陷阱"
- 如果两条概念容易混淆，点出区分要诀
- 一两句，不写长段落

### 格式

```
## {问题}
- category: {课程名}
- answer: |
  1. {得分点1}
  2. {得分点2}
- explanation: |
  {陷阱}
---
```

## 三、转换例子

卡材某条：
> 得分锚点：1. 构成 2. 单位 3. 来源——第三条是魏尔啸
> 易错陷阱：第三条错配给施莱登施旺

卡片：
> Q: 细胞学说的三条核心内容是什么？分别由谁提出？
> A: 1. 所有生物由细胞构成（施莱登&施旺，1838-1839）2. 细胞是生命活动的基本单位（同上）3. 细胞来自细胞（魏尔啸，1855）
> E: 第三条容易错配给施莱登施旺——魏尔啸是后来单独补上的，不是和施莱登施旺一起提的
