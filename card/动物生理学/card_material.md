# 闪卡原材料

## 标记：正反馈与负反馈的实例区分
- 为什么值得制卡：课堂中最明显卡点——学生倾向于用"目的"（止血=让身体恢复正常）而非"机制"（方向是往回拉还是往前推）来判断
- 关键得分点（≤3条）：
  - 负反馈：往回拉，维持稳态——如血压调节、体温调节
  - 正反馈：往前推，破坏原有平衡——如血液凝固、分娩
  - 判断法：只看方向不问目的——受控部分活动是向反方向改变（负）还是继续加强同方向（正）
- 常见错误：
  - 把凝血当负反馈（"凝血让身体恢复正常"）
  - 把血压调节当正反馈
- 来源：教材第1章 / 课堂推导

## 标记：四种跨膜转运方式的判断标准
- 为什么值得制卡：新概念，且学生首次判断葡萄糖转运时出错（说成了逆浓度主动转运）
- 关键得分点（≤3条）：
  - 简单扩散：顺浓度、不耗能、无需膜蛋白（O₂、CO₂）
  - 易化扩散：顺浓度、不耗能、需载体或通道蛋白（葡萄糖进多数细胞）
  - 主动转运：逆浓度、耗ATP、需离子泵（钠泵）
- 常见错误：
  - 葡萄糖进细胞=主动转运（实际多数是易化扩散）
  - 需要膜蛋白帮忙=耗能（不，膜蛋白帮了忙但没消耗ATP）
- 来源：教材第2章 / 课堂推导

## 标记：兴奋性的现代定义
- 为什么值得制卡：学生口误"动作定位"→"动作电位"，需要强化精确术语
- 关键得分点（≤3条）：
  - 经典定义：对刺激发生反应的能力
  - 现代定义：受刺激时产生动作电位的能力
  - 判断标准：只有能产生动作电位的组织才是可兴奋组织
- 常见错误：
  - "会反应就是有兴奋性"（忽略了生物电特征）
  - 兴奋=行为反应（忽略了电生理本质）
- 来源：教材第1章

## 标记：内环境稳态 = 动态相对稳定
- 为什么值得制卡：核心概念，学生理解正确但表述需要更精确
- 关键得分点（≤3条）：
  - 内环境 = 细胞外液（组织液、血浆、淋巴）
  - 稳态 ≠ 恒定不变，而是波动被控制在正常范围内
  - 稳态失调 → 疾病（发烧、糖尿病、尿毒症等）
- 常见错误：
  - 把稳态理解为"不变"
  - 混淆内环境（细胞外液）和血液
- 来源：教材第1章 / 课堂推导
