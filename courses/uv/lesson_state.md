## 已建立的结论

- conda 的 base 是默认环境，不切环境会导致不同项目的依赖混在一起
- uv 不给默认环境，需要每个项目有自己的 `.venv`（用 `uv venv` 创建）
- `uv pip install` 比 conda 安装快得多
- 虚拟环境就是一个给每个项目单独隔离的"小房间"

## 正在卡住的问题
