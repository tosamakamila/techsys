# 教学优化记录

课后群聊内部不输出，结构化写入。保留最近 5 条。同一观察 ≥3 次 → 升迁到 `learner_profile.md`。

## 格式

```
## 第N课 (日期)

| 维度 | 诊断详情 | 优化策略 |
|:---|:---|:---|
| 推理断层点 | 在[具体概念/步骤]处卡壳，未能自主推导 | 调整该知识点的过程拆解步长，增加前置支架 |
| 交互特征 | 出现N次含糊回答，[基础概念]掌握不牢 | 下次前置闪卡加入相关测试 |
| 策略有效性 | 使用[具体方法]后成功自主推导 | 保持该引导路径 |
```

禁止：小说式寒暄、动作描写、角色对话、"有效的/需调整的"散文化列表。
