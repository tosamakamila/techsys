# 教学优化记录（细胞生物学）

## 第3课 (2026-05-25) 绪论下——原核/古核/真核+病毒

| 维度 | 诊断详情 | 优化策略 |
|:---|:---|:---|
| 推理断层点 | 原核结构部分用户给出碎片化术语（"存在拟合"→"无核膜"→不完整句），属猜而非推；用户直接反馈"用比喻来引导"——连环追问在对方无具象抓手时等于猜谜 | S3（类比试探）优先级提到S4之前；卡住≥2次直接从比喻切入，不继续追问 |
| 推理断层点 | 革兰氏染色机制从"为什么G+染紫G-染红"到"青霉素差异"的推理链，用户单向回答无回推验证 | 下次类似对比机制→先让学生看完差异后用自己的话反推因果关系 |
| 交互特征 | 用户首次主动给出教学反馈（"用比喻来引导"）——这是重要的交互信号，说明用户对自己的学习状态有元认知判断 | 保持回应方式：被指正时立即认+改，不做防御性解释 |
| 交互特征 | 课堂后半段（病毒部分）用户回答明显比前半段完整——与比喻引导切换时间点吻合 | 证据积累：比喻引导效果显著，保持为默认S3策略 |
| 策略有效性 | 图书馆比喻（核膜=图书馆墙，转录翻译=阅读加工分房）效果立竿见影——用户从碎片化术语切换到完整参与 | 将"先用比喻建立具象锚点"升格为默认引导策略 |
| 策略有效性 | 真核三大体系用"猜名字"方式（"这套叫什么？"）——用户能自主命名两套（生物膜、骨架），参与度高 | 命名权交出的时机准则：概念已通过具体例子铺垫→让对方猜名称 |
| 概念密度 | 四线并推（原核+古核+真核+病毒）后用户无法自主判断薄弱点（问"哪个点虚"→回答"不知道"） | 下次多主题课程：课末逐框确认而非开放式提问；或每主题结束后即时检验 |
