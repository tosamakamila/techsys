# 引导点预案模板

在 `transformed/` 教案文件中，为核心概念嵌入引导点标注。一节课不超过 3 个。HTML 注释是机器可读的结构标记，正文是灵的预案参考。

## 格式

```markdown
## 引导点: [概念名]

<!-- gp: Nxxx -->              ← 对应 knowledge_map 节点ID（必填）
<!-- gp: prereq Nxxx -->       ← 前置知识节点（可选，多个则多行）
<!-- gp: type 对比引导 -->     ← 对比引导/过程拆解/推到极端/反例检验/场景假设（必填）

**铺路：** [1-2句，总结已建立的前提。把缺口的方向指清楚]

**主问题：** [开放引导问题。不能是是非题。必须有推理支点]

**预期回答分支：**

→ ✅ 方向正确（[信号：比如"提到分裂图像/否定了自然发生"]）：
   "[灵的回应——顺着答案推下一步，不评分]"

→ ⚠️ 部分正确（[信号：比如"说了分裂但没排除其他可能"]）：
   "[灵的回应——指出缺口方向，给半步]"

→ ❌ 方向偏离（[信号：比如"回答细胞怎么分裂的细节"]）：
   "[灵的回应——笑着拉回，换角度重新框定]"

**过渡：** [无论哪条路径，最终桥接到下一个概念。1句话]
```

## 示例：细胞学说第三条

```markdown
## 引导点: "细胞来自细胞"

<!-- gp: N003 -->
<!-- gp: prereq N002 -->
<!-- gp: type 对比引导 -->

**铺路：** 前面两条说了两件事——(1)所有生物由细胞构成，(2)细胞是生命活动的基本单位。这两条都描述了"细胞是什么"。第三条"细胞来自细胞"在意的不是"是什么"，是"从哪来的"。

**主问题：** 魏尔啸1855年看到伤口边缘有新细胞——他凭什么断定这些细胞是旧细胞分裂来的，而不是从伤口液体里凭空冒出来的？

**预期回答分支：**

→ ✅ 方向正确（提到分裂图像或逻辑排除"凭空出现"）：
   "对——显微镜下确实看到了分裂中的细胞。但魏尔啸说这句话时还有一个逻辑前提——他必须先否定什么？"

→ ⚠️ 部分正确（提到分裂但没排除其他可能）：
   "分裂能解释一部分。但他说的是'所有'细胞——凭什么排除有些细胞可以从无到有？当时自然发生说还很流行。"

→ ❌ 方向偏离（回答成分裂的细节机制）：
   "噗——那是'怎么产生'的。我更好奇的是——看伤口长新细胞时，魏尔啸脑子里在反对谁？"

**过渡：** 否定自然发生说 → 所有细胞来自已有细胞 → 自然引出：那第一个细胞怎么来的？
```

## 使用规则

1. **预案不是剧本。** 灵上课时不逐字读——这是备课时存入的判断框架
2. 学习者回答不在三个分支中 → 灵按 CLAUDE.md 引导法则临场应对
3. 一节课不超过 3 个引导点——只标最关键的概念转折处
4. 灵开口仍是口语化、带颗粒感的，不是教案腔
5. 引导点由备课阶段填充，上课前检查：教案有 `<!-- gp: -->` 则参考，无则灵临场引导
