# 长教材分片与教案模板

> ⚠️ **已弃用。** 教材即教案，教师直接按 `materials/` 原文推进，不生成 `transformed/` 二次加工教案。本文件仅供历史参考。

用于处理长教材：切分为适合一节课的片段，按需生成教案。

## 核心原则

1. 长教材不全篇读入，每次只读取一个章节或一组紧密关联的知识点
2. 每次处理后更新 `reading_plan.md`
3. 教案必须输出到 `transformed/`，存入后直接使用。命名规则：`{片段ID}_{中文主题}.md`（如 `L002_显微镜与细胞观察.md`）
4. 只有需要回顾前文时才读旧教案或摘要
5. 教材缺漏→写入 `book_revision_notes.md` 教师补充内容

## 分片粒度

- **按章切分（默认）**：教材有清晰章节结构，每章内容量适中
- **按小节切分**：理论密集型（单章 >3000 字），按知识点拆分
- **动态分片（推荐）**：初始按章，上课卡住再拆。不要一开始切太细
- **困难章预警**：行数 >350 且标记"困难"→首次至少 3 课时

## reading_plan.md

按 `course_folder_protocol.md` 长教材读取规则初始化。状态值：未处理 / 已生成教案 / 已上课 / 需复习 / 跳过。

## 教案格式（按需生成）

教案至少包含：

```markdown
# 教案：片段ID - 主题名

## 结束点
- 本堂课只处理：
- 不处理：
- 下课条件：

## 核心问题
用一个主问题统领本节课。

## 知识边界
- 必须掌握：
- 只需有印象：
- 暂时不讲：

## 问题链（递进，一次一个台阶）
### Q1. 
- 提问目的：
- 理想回答 / 偏了追问 / 卡住提示：
- 本轮小结：

## 典型误区
- 错误说法 / 为什么容易这么想 / 如何纠正：

## 课堂检查点（2-4 个）

## 下课总结标签（1-3 个短标签）
```

## 复习课处理

片段标记"需复习"时：读旧教案或原文 → 查看 `lesson_state.yaml` 卡点 → 从卡点重新开始（换例子/角度，不覆盖全课）。详见 `review_lesson.md`。

## 课后更新

见 `after_class_update.md`。

## 引导点标注（可选）

对 knowledge_map 中标记为 `strong` 依赖的关键节点，可在教案中嵌入引导点预案。格式见 `guidance_point_template.md`。一节课不超过 3 个，仅标注最关键的概念转折处。
