# 分课程文件夹协议

本文件为参考说明。启动加载见 CLAUDE.md「上课启动流程」和「课程匹配」。日常上课不再读取此文件。

## 课程文件夹结构

- `course.md`：名称、别名、目标、教材来源（仅建课/推进新课章时读）
- `progress.md`：历史归档（仅需历史时读）
- `lesson_state.yaml`：课堂持久状态（课后读写，课堂不读）
- `lesson_entry.yaml`：课堂入口（fragment/interrupted_at/stuck/entry_line，上课时读）
- `learner_profile.md`：本课程的学习者画像（每课必读）
- `reading_plan.md`：分片计划（仅推进新片段时读）
- `materials/`：原始教材
- `transformed/`：单课教案
- `notes/`：用户笔记
- `book_revision_notes.md`：教材修订记录

## 课后更新

按 `after_class_update.md` 执行。
