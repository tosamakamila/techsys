# 第三章 血液

## 🔑 一、 核心机制与关键考点

### 1. 血液的组成

- **核心原理**：血液由血浆和悬浮于血浆中的血细胞组成。
- **血浆成分**：91.5%水、1.5%盐/糖/脂肪/氨基酸、7%蛋白质
- **有形成分**：1%白细胞和血小板、99%红细胞
- **离心分层**：上层浅黄色血浆、下层深红色红细胞、中间白色薄层白细胞和血小板

### 2. 血液的主要机能

| 功能 | 说明 |
| --- | --- |
| 营养功能 | 血浆蛋白质起营养储备作用，单核巨噬细胞吞饮血浆蛋白→分解为氨基酸→供给其他细胞 |
| 运输功能 | 运送O₂、营养物质、激素至组织细胞；运送CO₂、尿素、尿酸、肌酐等至排泄器官 |
| 维持内环境稳定 | 缓冲物质维持酸碱平衡；胶体渗透压维持血管内外水平衡；晶体渗透压维持细胞内外水平衡；水分吸热散热维持体温恒定 |
| 参与体液调节 | 激素由血液运送至靶细胞 |
| 防御和保护功能 | 白细胞吞噬；抗毒素、溶菌素对抗细菌毒素；凝血因子、抗凝物质、纤溶系统起保护作用 |

### 3. 血液理化特性

| 特性 | 内容 |
| --- | --- |
| 颜色 | 动脉血鲜红色（Hb含氧多），静脉血暗红色（Hb含氧少） |
| 质量密度 | 全血1.050～1.060×10³kg/m³；红细胞1.070～1.090×10³；血浆1.025～1.030×10³ |
| 黏滞性 | 全血为水的4～5倍（取决于红细胞数），血浆为水的1.6～2.4倍（取决于血浆蛋白含量）；血流缓慢时红细胞叠连→黏滞性↑ |
| pH | 弱碱性7.2～7.5 |

### 4. 晶体渗透压 vs 胶体渗透压

| 对比维度 | 晶体渗透压 | 胶体渗透压 |
| --- | --- | --- |
| 来源 | 晶体物质（80%来自Na⁺和Cl⁻） | 血浆蛋白（主要来自白蛋白） |
| 数值 | 占血浆渗透压绝大部分 | 很小，不超过3.3kPa（25mmHg） |
| 毛细血管壁通透性 | 晶体物质可自由通过→血浆与组织液晶体渗透压基本相等 | 血浆蛋白不能透过→血浆胶体渗透压始终高于组织液 |
| 生理意义 | 维持细胞内外水平衡和细胞正常形态 | 维持血管内外水平衡 |

### 5. 等渗溶液 vs 等张溶液

| 对比维度 | 等渗溶液 | 等张溶液 |
| --- | --- | --- |
| 定义 | 与血浆渗透压相等的溶液 | 能使红细胞保持正常体积和形状的盐溶液 |
| 举例 | 0.9%NaCl溶液、1.9%尿素溶液 | 0.9%NaCl溶液 |
| 关系 | 等渗不一定等张 | 等张一定等渗 |
| 关键区别 | — | 尿素能自由通过细胞膜→红细胞在其中溶血→等渗但非等张 |

### 6. 血浆酸碱缓冲系统

- **核心原理**：血液中缓冲对维持pH相对稳定，NaHCO₃/H₂CO₃缓冲能力最强。
- **血浆缓冲对**：NaHCO₃/H₂CO₃（20:1，最强）、Na₂HPO₄/NaH₂PO₄、蛋白质钠盐/蛋白质
- **红细胞内缓冲对**：血红蛋白钾盐/血红蛋白、氧合血红蛋白钾盐/氧合血红蛋白、K₂HPO₄/KH₂PO₄、KHCO₃/H₂CO₃
- **酸中毒缓冲**：HL + NaHCO₃ → NaL + H₂CO₃ → H₂O + CO₂（肺排出）
- **碱中毒缓冲**：Na₂CO₃ + H₂CO₃ → 2NaHCO₃（肾脏排出）
- **关键影响因素/注意点**：
    - 血液缓冲对的作用必须与呼吸系统和泌尿系统配合
    - 挥发性酸直接通过肺脏排出，非挥发性酸随尿排出
    - 挥发性酸或碱增加且不能及时排出→呼吸性酸中毒或碱中毒

### 7. 红细胞的形态与功能

- **核心原理**：红细胞是血液中数量最多的血细胞，主要功能是运输O₂和CO₂。
- **形态**：
    - 鱼类、两栖类、爬行类、鸟类：扁卵圆形，有细胞核
    - 哺乳动物：双凹圆盘形，无细胞核，直径7～10μm
- **双凹圆盘形优势**：
    1. 表面积/体积之比较球形大→气体扩散距离缩短
    2. 有利于可塑性变形→通过比其直径还小的毛细血管
- **功能**：
    1. 运输O₂和CO₂（主要功能，由血红蛋白完成）
    2. 对酸碱物质有缓冲作用
    3. 免疫功能（表面补体C3受体→吸附抗原-补体复合物→由吞噬细胞消灭）
- **关键影响因素/注意点**：
    - 红细胞缺乏线粒体→厌氧代谢产生ATP→运输O₂不消耗O₂→提高运输效率
    - 红细胞数量：幼年>成年，雄性>雌性，高海拔>低海拔

### 8. 血红蛋白的结构与功能

- **核心原理**：血红蛋白（Hb）由1个珠蛋白和4个血红素组成，相对分子质量约68000。
- **结构**：
    - 珠蛋白：两条α链+两条β链
    - 血红素：4个吡咯基形成环，Fe²⁺在中心
    - Fe²⁺与O₂结合实现运输
- **携O₂阻断因素**：
    - CO与Hb亲和力高于O₂→取代O₂→形成碳氧血红蛋白（HbCO）
    - 亚硝酸盐等氧化剂→Fe²⁺→Fe³⁺→形成高铁血红蛋白（metHb）→O₂不能释放

### 9. 红细胞可塑性变形的影响因素

| 因素 | 与变形能力的关系 | 机制 |
| --- | --- | --- |
| 表面积/体积比值 | 正比 | 双凹圆盘形比值适宜→变形能力强 |
| 膜流动性、弹性 | 正比 | 胆固醇/磷脂比值、饱和/不饱和脂肪酸比值影响膜流动性 |
| 红细胞内黏度 | 反比 | 血红蛋白变形或浓度增高→内黏度↑→变形能力↓ |
| 胞浆Ca²⁺浓度 | 反比 | Ca²⁺↑→胞浆由溶胶变凝胶→内黏度↑→变形能力↓ |

### 10. 红细胞悬浮稳定性与血沉

- **核心原理**：红细胞悬浮稳定性是红细胞重力和与血浆之间摩擦力相互作用的结果。
- **叠连的影响**：
    - 红细胞叠连→总外表面积/容积之比减小→摩擦力减小→下沉加速
    - 促使叠连的因素取决于**血浆性质**（非红细胞本身）
- **加速血沉**：球蛋白↑、纤维蛋白原↑、胆固醇↑
- **减慢血沉**：白蛋白↑、卵磷脂↑
- **血沉加快的疾病**：急性感染、风湿、结核

### 11. 红细胞的生成过程

- **核心原理**：红细胞起源于造血干细胞，经历三个阶段发育成熟。
- **造血器官变迁**：胚胎早期（中胚层）→卵黄囊、肝脏、脾脏→胚胎中期（骨髓开始造血）→出生后（几乎完全依靠骨髓造血）
- **造血三阶段**：
    1. **造血干细胞阶段**：自我复制保持数量稳定+分化形成各系定向祖细胞
    2. **定向祖细胞阶段**：红系祖细胞（CFU-E）、粒-单核系祖细胞（CFU-GM）、巨核系祖细胞（CFU-MK）、TB淋巴系祖细胞（CFU-TB）
    3. **前体细胞阶段**：形态上可辨认的各系幼稚细胞→分化成熟为终末血细胞
- **红细胞发育阶段**：定向红系祖细胞→原始红细胞→早幼红细胞→中幼红细胞→晚幼红细胞→网织红细胞→成熟红细胞
- **关键影响因素/注意点**：
    - 造血细胞发育过程中：自我复制能力逐渐降低，分化增殖能力逐渐增强，细胞数量逐步增大
    - 成年后造血骨髓仅存在于脊椎骨、肋骨、胸骨、颅骨和长骨近端

### 12. 红细胞生成的调节

- **核心原理**：促红细胞生成素（EPO）是调节红细胞生成的主要体液因素。
- **EPO**：
    - 主要产生于肾脏
    - 作用：促进红系祖细胞增殖分化为原红细胞
    - 缺O₂→肾脏产生EPO↑→红细胞生成↑
- **其他调节因素**：
    - 雄激素：刺激EPO产生，也直接刺激骨髓造血
    - 雌激素：抑制红细胞生成
    - 甲状腺激素、生长激素也可促进红细胞生成

### 13. 白细胞分类与功能

| 白细胞类型 | 主要功能 | 关键特点 |
| --- | --- | --- |
| 中性粒细胞 | 吞噬外来微生物和异物，非特异性细胞免疫第一线 | 数量最多；血管内停留6～8h；活跃变形、高度趋化、强吞噬；释放花生四烯酸→血栓素和前列腺素 |
| 嗜碱性粒细胞 | 参与过敏反应、脂肪代谢 | 释放组胺（毛细血管扩张、通透性增加，短暂）、过敏性慢反应物质（缓慢持久，增加通透性，加强平滑肌收缩）、肝素（抗凝血、激活脂肪分解）、过敏性趋化因子 |
| 嗜酸性粒细胞 | 限制过敏反应、参与对蠕虫免疫 | 释放前列腺素E抑制嗜碱性粒细胞；吞噬嗜碱性粒细胞颗粒；释放组胺酶破坏组胺；借Fc受体和C3受体黏着蠕虫→释放碱性蛋白和过氧化物酶损伤蠕虫；数目有昼夜波动（与糖皮质激素有关） |
| 单核细胞 | 吞噬作用、抗原提呈 | 胞体大，吞噬能力最强；迁移至肝脾淋巴结后转变为巨噬细胞；产生细胞因子和促生长因子；免疫初期提呈抗原给淋巴细胞 |
| 淋巴细胞 | 免疫应答核心 | T细胞（80%～90%）→细胞免疫；B细胞→体液免疫 |

### 15. 血小板的生理特性

| 特性 | 内容 |
| --- | --- |
| 黏附 | 与血管内皮下成分结合；参与蛋白质：血小板质膜GP、内皮下胶原纤维、vWF；黏附引起花生四烯酸→TXA₂（促聚集和促血管收缩） |
| 聚集 | 第一时相：迅速、外源性ADP引起、可逆；第二时相：缓慢、内源性ADP引起、不可逆 |
| 释放反应 | 激动剂（ADP、TXA₂、胶原、凝血酶）刺激下释放ADP、5-HT、儿茶酚胺、Ca²⁺、PF3等 |
| 收缩 | 收缩蛋白收缩→血凝块回缩、血栓硬化→有利于止血 |

### 16. 血小板的生理功能

- **核心原理**：血小板主要功能是维持血管内皮完整性、参与生理性止血和血液凝固。
- **生理性止血三步骤**：
    1. **血管收缩**：神经反射性收缩+内皮细胞和血小板释放缩血管物质（5-HT、ADP、TXA₂、内皮素）→封闭创口
    2. **血小板止血栓形成**：血小板黏附、聚集→形成松软止血栓→初步止血
    3. **血凝块形成**：激活凝血系统→纤维蛋白凝块→加固止血栓→有效止血
- **参与凝血**：
    - 提供磷脂表面（PF3）→凝血酶原激活加速20000倍
    - PF2促进纤维蛋白原→纤维蛋白单体
    - PF4抗肝素作用→有利于凝血酶生成
- **保持血管内皮完整性**：血小板融合进入内皮细胞→填补脱落内皮→促进修复；血小板减少→血管脆性增加→易出血

### 17. 凝血因子要点

- **核心原理**：凝血因子共14种（因子Ⅰ～ⅩⅢ，加上前激肽释放酶），除Ca²⁺和磷脂外均为蛋白质。
- **关键特征**：
    - 因子Ⅲ（组织因子TF）存在于组织中，其他均存在于血浆中
    - 多数凝血因子由肝脏合成
    - FⅡ、FⅦ、FⅨ、FⅩ的生成必须有维生素K
    - 凝血因子Ⅱ、Ⅶ、Ⅸ、Ⅹ、Ⅺ、Ⅻ及前激肽释放酶均为丝氨酸蛋白酶
    - 以酶原形式存在→激活后代号右下角加"a"
    - 血清中不存在的因子：纤维蛋白原（FⅠ）、FⅤ、FⅧ（被消耗）

### 18. 凝血过程三阶段

**凝血第Ⅰ阶段——凝血酶原复合物形成**

| 对比维度 | 内源性凝血途径 | 外源性凝血途径（组织因子途径） |
| --- | --- | --- |
| 启动方式 | 血液与带负电荷异物表面接触 | 组织因子FⅢ（TF）暴露 |
| 始动因子 | FⅫ被激活为FⅫa | TF与FⅦ/FⅦa结合 |
| 参与因子 | FⅫ、FⅪ、FⅨ、FⅧ、Ca²⁺、磷脂 | TF、FⅦ、Ca²⁺ |
| 速度 | 较慢（FⅨa+FⅧa激活FⅩ十分缓慢，FⅧ可使反应速度提高20倍） | 较快（正反馈效应：FXa激活FⅦ→更多FXa） |
| 交汇点 | FXa生成后进入共同通路 | FXa生成后进入共同通路 |

- **关键影响因素/注意点**：
    - FⅧ是重要辅助因子（非蛋白酶），存在于血浆中，通常与vWF以非共价键结合成复合物
    - 修正后的瀑布学说：体内凝血几乎都由外源性途径启动
    - TFPI（组织因子途径抑制物）使外源性途径生成凝血酶极少→微量凝血酶激活血小板和FⅤ、FⅧ、FⅨ、FⅪ→加强内源性途径→生成足量凝血酶

**凝血第Ⅱ阶段——凝血酶原→凝血酶**
- FXa-FVa-Ca²⁺-磷脂形成凝血酶原复合物→激活凝血酶原（FⅡ）为凝血酶（FⅡa）
- 凝血酶的多功能：分解纤维蛋白原→纤维蛋白单体；激活FⅤ、FⅦ、FⅧ、FⅨ、FⅩ、FⅩⅢ、血小板；灭活FVa、FⅧa（制约凝血）

**凝血第Ⅲ阶段——纤维蛋白原→纤维蛋白**
- 凝血酶作用→α、β链断裂脱下小肽→纤维蛋白单体
- FXⅢa和Ca²⁺作用下→纤维蛋白单体聚合→不溶性交联纤维蛋白多聚体凝块

### 19. 抗凝系统

| 抗凝物质 | 来源 | 作用机制 |
| --- | --- | --- |
| 抗凝血酶Ⅲ | 肝细胞和血管内皮细胞分泌 | 与FⅦa、FⅨa、FⅩa、FⅪa、FⅫa和凝血酶的丝氨酸残基结合→封闭活性位点→失活；与肝素结合后抗凝活性增加约2000倍 |
| 肝素 | 肥大细胞和嗜碱性粒细胞产生 | ①与抗凝血酶Ⅲ结合→亲和力增强约100倍；与肝素辅助因子Ⅱ结合→灭活凝血酶速度加快1000倍；②刺激血管内皮细胞释放TFPI；③提高蛋白质C活性；④刺激释放纤溶酶原激活物 |
| 蛋白质C | 肝脏合成（维生素K依赖因子） | 凝血酶激活后：①灭活FVa、FⅧa；②阻碍FXa与血小板磷脂结合；③刺激纤溶酶原激活物释放→增强纤溶 |
| TFPI | 小血管内皮细胞 | 体内主要生理性抗凝物质；①K2与FXa结合→直接抑制FXa→TFPI变构；②变构TFPI与FⅦa-TF结合→形成四聚体→灭活FⅦa-TF复合物→负反馈抑制外源性凝血 |

### 20. 纤维蛋白溶解系统

- **核心原理**：纤溶是纤维蛋白被分解液化的过程，与凝血对立统一。
- **纤溶两阶段**：
    1. **纤溶酶原激活**：
       - 外源性激活途径：血管激活物（小血管内皮细胞合成）、组织激活物
       - 内源性激活途径：FⅫa、激肽释放酶
    2. **纤维蛋白降解**：纤溶酶水解肽链上赖氨酸-精氨酸键→纤维蛋白降解产物（可溶性小肽）
- **纤溶酶特点**：血浆中活性最强的蛋白酶，特异性较小
    - 水解纤维蛋白原/纤维蛋白、凝血酶、FⅤ、FⅧ、FⅫa
    - 促使血小板聚集和释放5-HT、ADP
    - 激活补体系统
- **纤溶抑制物**：PAI-1（抑制tPA和尿激酶）、补体C₁抑制物、α₂-巨球蛋白、蛋白酶C抑制物、抗凝血酶Ⅲ
- **关键影响因素/注意点**：
    - 纤维蛋白降解产物一般不能再凝固，部分还有抗凝作用
    - 凝血与纤溶平衡破坏→血栓形成或出血性疾病
    - 纤溶抑制物既能抑制纤溶又能抑制凝血→使凝血和纤溶局限在创伤局部

### 21. ABO血型系统

| 血型 | 红细胞膜凝集原 | 血清凝集素 |
| --- | --- | --- |
| A型（A₁） | A+A₁ | 抗B |
| A型（A₂） | A | 抗B+抗A₁ |
| B型 | B | 抗A |
| AB型（A₁B） | A+A₁+B | 无 |
| AB型（A₂B） | A+B | 抗A₁ |
| O型 | 无A，无B | 抗A+抗B |

- **关键影响因素/注意点**：
    - H抗原是A、B抗原的结构基础，4种血型红细胞上都含H抗原（抗原性较弱）
    - 血清中不含对抗本身红细胞凝集原的凝集素
    - 遗传：受复等位基因（Iᴬ、Iᴮ、i）控制，基因座在9q34上；Iᴬ或Iᴮ对i显性，Iᴬ与Iᴮ共显性

### 22. Rh血型系统

- **核心原理**：Rh血型抗原多达40余种，与临床关系密切的是D、E、C、c、e 5种，D抗原性最强。
- **分类**：含D抗原→Rh阳性；缺乏D抗原→Rh阴性
- **关键影响因素/注意点**：
    - 人类血清中不存在抗Rh天然抗体
    - Rh阴性接受Rh阳性血液→产生抗Rh抗体→第二次输Rh阳性血→凝集反应
    - Rh阴性母亲怀Rh阳性胎儿→胎儿红细胞或D抗原通过胎盘进入母体→母体产生抗D抗体→抗体透过胎盘进入胎儿→新生儿溶血性贫血
    - 汉族99%以上为Rh阳性，苗族12.3%为Rh阴性，塔塔尔族15.8%为Rh阴性

### 23. 输血原则

- **核心原理**：必须保证供血者与受血者血型相合，防止凝集反应。
- **输血原则**：
    1. 首先鉴定ABO血型→保证相合
    2. 生育年龄妇女和反复输血患者→还须Rh血型相合
    3. 即使ABO血型相同→也须交叉配血试验
- **交叉配血试验结果**：
    - 主侧+次侧均无凝集→配血相合→可以输血
    - 主侧有凝集→配血不合→不能输血
    - 主侧无凝集+次侧有凝集→应急情况下可输，但不宜太快太多
- **关键影响因素/注意点**：
    - O型血不是"万能供血者"——血浆中抗A和抗B凝集素可凝集其他血型红细胞
    - AB型不是"万能受血者"
    - 提倡成分输血→提高疗效、减少不良反应、节约血源

### 24. 失血的恢复

| 失血量 | 后果 | 恢复机制 |
| --- | --- | --- |
| ≤10% | 一般不影响健康 | 血浆水分和无机物1～2h内由组织液补偿；血浆蛋白由肝脏加速合成；血细胞由储备血液暂时补充，造血器官缓慢生成 |
| >20% | 机体活动明显障碍 | — |
| >30% | 可能危及生命 | — |

## 🧬 四、 核心名词解释（严格原文）

* **【血细胞比容】**：利用离心方法可测出红细胞在全血中所占的体积分数，即血细胞比容。不同种类动物的红细胞比容不同。

* **【血量】**：动物体内血浆量和血细胞量的总和，即血液的总量称为血量。哺乳动物占体重的5%～10%，正常成年人的血量为体重的7%～8%。

* **【循环血量】**：在安静状态下，大部分血液在心血管内流动，这部分血量称为循环血量。

* **【储备血量】**：少部分滞留在肝脏、肺脏、脾脏、皮下静脉丛和皮肤等处，称为储备血量。在剧烈运动、情绪激动或失血时，储备血量释放出来补充循环血量的相对不足。

* **【黏滞性】**：液体流动时，由于其内部分子摩擦而产生的阻力，以致流动缓慢并表现出黏着的特性，称为黏滞性。血液的黏滞性为水的4～5倍，血浆的黏滞性为水的1.6～2.4倍。

* **【渗透压】**：渗透压是指溶液中的溶质颗粒通过半透膜吸取水分子的一种力量。渗透压的大小由单位体积溶液中溶质颗粒的数目决定，而与其溶质分子种类和溶质颗粒大小无关。

* **【晶体渗透压】**：由晶体物质所形成的渗透压称为晶体渗透压，其中80%来自Na⁺和Cl⁻。

* **【胶体渗透压】**：由血浆蛋白形成的渗透压称为胶体渗透压，通常不超过3.3kPa（25mmHg）。血浆胶体渗透压主要来自白蛋白。

* **【等渗溶液】**：与血浆渗透压相等的溶液称为等渗溶液（如0.9%NaCl溶液）。

* **【等张溶液】**：能使悬浮于其中的红细胞保持正常体积和形状的盐溶液，称为等张溶液。1.9%的尿素溶液为等渗溶液，但并非等张溶液。

* **【碱储】**：NaHCO₃的浓度变化影响着血液对酸碱度的缓冲能力，因此血液中NaHCO₃的含量（或浓度）称为碱储。血浆中NaHCO₃和H₂CO₃的比值通常为20:1。

* **【溶血】**：如果将红细胞置于低渗的NaCl溶液中，水分进入红细胞内使之膨胀，严重时红细胞因过度膨胀而破裂，血红蛋白逸出，这一现象称为红细胞溶解，简称溶血。

* **【红细胞渗透脆性】**：红细胞在低渗溶液中发生膨胀、破裂和溶血的特性，称为渗透脆性。渗透脆性越大的红细胞，对低渗盐溶液的抵抗力越小。衰老的红细胞渗透脆性大，初成熟的红细胞渗透脆性小。

* **【红细胞机械性脆性】**：由于物理的原因（碰撞、挤压等）而引起的红细胞破裂称为机械性脆性。

* **【可塑性变形】**：红细胞经常要挤过口径比它小的毛细血管和血窦孔隙，这时红细胞发生变形，呈子弹或降落伞状，然后又恢复原形，这种变形称为可塑性变形。

* **【红细胞悬浮稳定性】**：红细胞的质量密度虽然比血浆大，但在血浆中却能够保持悬浮状态而不易下沉，这一特性称为红细胞的悬浮稳定性。

* **【红细胞沉降率（血沉，ESR）】**：将抗凝血置于有刻度的血沉管中，红细胞下沉的距离表示红细胞沉降的速度，称为红细胞沉降率或血沉。ESR越小表示红细胞悬浮稳定性越大。

* **【红细胞叠连】**：某些疾病可使红细胞相互叠连起来，其总的外表面积与容积之比减小，因而摩擦力减小，下沉速度加快。促使红细胞发生叠连的因素主要决定于血浆性质，而不是红细胞本身。

* **【促红细胞生成素（EPO）】**：促红细胞生成素主要产生于肾脏，是调节红细胞生成的主要体液因素。

* **【血细胞渗出】**：除淋巴细胞外，所有白细胞都能伸出伪足做变形运动，凭借这种运动白细胞得以穿过血管壁，这一过程称为血细胞渗出。

* **【趋化性】**：白细胞具有向某些化学物质游走的特性，称为趋化性。体内具有趋化作用的物质包括机体细胞的降解产物、抗原-抗体复合物、细菌毒素和细菌等。

* **【吞噬作用】**：白细胞可循着化学物质的浓度梯度游走到这些物质的周围，把异物包围起来并吞入胞浆内，此过程称为吞噬作用。

* **【血小板黏附】**：血小板黏附是指它与血管内皮下成分结合的过程。当血管内皮损伤而暴露胶原组织时，立即引起血小板的黏着。

* **【血小板聚集】**：血小板彼此之间互相黏附、聚合成团的过程，称为血小板聚集。第一时相为可逆聚集（外源性ADP引起），第二时相为不可逆聚集（内源性ADP引起）。

* **【血液凝固（血凝）】**：血液凝固是指血液由流动的溶胶状态转变成不能流动的凝胶状态的过程。血浆中的可溶性纤维蛋白原转变为不溶的纤维蛋白，并交织成网，网罗许多血细胞。

* **【血清】**：血液凝固后1～2h，血块发生回缩，析出淡黄色的液体，称为血清。血清和血浆的区别是血清去除了纤维蛋白原和少量参与凝血的血浆蛋白，增加了血小板释放的物质。

* **【凝血因子】**：血浆与组织中直接参与血液凝固的物质，统称为凝血因子。除Ca²⁺（因子Ⅳ）与磷脂外，其余的凝血因子均为蛋白质。

* **【内源性凝血途径】**：完全依靠血浆中的凝血因子逐步激活FX，称为内源性激活，其发生的凝血称为内源性凝血途径。

* **【外源性凝血途径（组织因子途径）】**：依靠血管外组织释放的组织因子FⅢ（TF）来参与激活FX，称为外源性激活，其发生的凝血称为外源性激活途径。

* **【纤维蛋白溶解（纤溶）】**：血液凝固过程中形成的纤维蛋白被分解、液化发生溶解的过程，称为纤维蛋白溶解，简称纤溶。

* **【凝集】**：将血型不相容的两个个体的血滴放在玻片上混合，其中的红细胞即聚集成团，这种现象称为凝集，是一种免疫现象。

* **【凝集原】**：凝集原的特异性取决于镶嵌在红细胞膜上的特异性糖蛋白或糖脂，它们在凝集反应中起抗原作用，因而称为凝集原。

* **【凝集素】**：将与红细胞膜上的凝集原起反应的特异抗体称为凝集素。凝集素是溶解在血浆的γ-球蛋白。

* **【交叉配血试验】**：把供血者的红细胞与受血者的血清进行配合试验称为交叉配血主侧；把受血者的红细胞与供血者的血清作配合试验称为交叉配血次侧。

* **【成分输血】**：把人血中的各种有效成分，如红细胞、粒细胞、血小板和血浆分别制备成高纯度或高浓度的制品再输入，称为成分输血。
