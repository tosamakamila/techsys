# 第六章 消化和吸收

## 🔑 一、 核心机制与关键考点

### 1. 三种消化方式及其关系

| 消化方式 | 定义 | 主要部位 | 特点 |
| --- | --- | --- | --- |
| 机械性消化 | 通过咀嚼和消化道肌肉舒缩完成 | 口腔、胃肠道 | 磨碎食物，与消化液混合，向远端移送 |
| 化学性消化 | 通过消化腺分泌的消化液完成 | 胃、小肠 | 消化酶将大分子分解为小分子 |
| 微生物消化 | 由消化道内微生物完成 | 反刍动物瘤胃、单胃动物盲肠 | 消化纤维素、半纤维素等高分子糖类 |

**三种消化方式的关系**：相互依存、相互配合、同时进行。机械性消化为化学性和微生物消化创造条件；化学性和微生物消化又影响机械性消化。

### 2. 消化道平滑肌的一般特性

| 特性 | 说明 |
| --- | --- |
| 兴奋性较低，收缩缓慢 | 肌质网不发达，Ca²⁺储备少，收缩潜伏期、收缩期和舒张期比骨骼肌长 |
| 富有伸展性 | 无Z线、M线和肌小节之分，可比原初长度增加几倍 |
| 持续的紧张性 | 经常保持微弱持续收缩状态，使管腔内保持一定压力，维持消化道形状和位置 |
| 自动节律性 | 某些特殊平滑肌细胞能自动产生节律性动作电位，离体后仍能缓慢节律运动 |
| 对某些理化刺激较敏感 | 对化学、温度和牵张刺激敏感，对电刺激不敏感 |

### 3. 消化道平滑肌的电生理特性

| 电活动 | 特点 | 生理意义 |
| --- | --- | --- |
| 静息电位 | 不稳定，波动大，-60～-50mV，主要取决于K⁺平衡电位 | 基础状态 |
| 慢波电位 | 波幅10～30mV，持续1～4s，肌源性，不依赖神经 | 起步电位，决定胃肠蠕动的方向、节律和速度；本身不引起肌肉收缩 |
| 动作电位 | 峰电位上升慢、持续时间长、幅度低；主要依赖Ca²⁺内流；频率越高收缩幅度越大 | 引起肌肉收缩 |

**关键**：慢波→接近阈电位→动作电位→肌肉收缩。慢波是收缩节律的控制波。

### 4. 消化液的主要功能

1. 改变消化道内的pH，以适应消化酶活性的需要
2. 将复杂的食物成分分解为简单的、可被吸收的小分子物质
3. 稀释食物或消化产物，调节消化道内容物的渗透压，便于吸收
4. 通过分泌黏液、抗体和大量液体，保护消化道黏膜

### 5. 消化道功能的神经调节

**外来神经系统**：
- **副交感神经（迷走神经为主）**：兴奋→胃肠运动增强、消化腺分泌增加
- **交感神经**：兴奋→胃肠运动抑制、消化腺分泌减少

**内在神经丛（壁内神经丛）**：
- **肌间神经丛（欧氏神经丛）**：位于纵行肌和环行肌之间，主要支配平滑肌收缩
  - 兴奋性神经元：递质为乙酰胆碱和P物质
  - 抑制性神经元：递质为VIP和一氧化氮
- **黏膜下神经丛（麦氏神经丛）**：分布在黏膜下，支配腺体分泌

**关键**：切断外来神经后，内在神经丛仍可通过局部反射调节消化管运动和腺体分泌。

### 6. 胃肠激素的三大生理功能

1. **调节消化道的运动和消化腺的分泌**
2. **调节其他激素的释放**（如抑胃肽刺激胰岛素分泌，防止血糖升高过快）
3. **营养作用**（如胃泌素促进胃和十二指肠黏膜蛋白质合成和生长；缩胆囊素促进胰腺外分泌组织生长）

### 7. 主要胃肠激素一览

| 激素 | 分泌部位 | 主要作用 | 释放因素 |
| --- | --- | --- | --- |
| 胃泌素 | 胃幽门腺及十二指肠G细胞 | 促进胃酸分泌、胃运动和胃黏膜生长 | 蛋白质消化产物；高pH；迷走神经兴奋 |
| 胰泌素 | 十二指肠S细胞 | 促进胰腺分泌碳酸氢盐 | 十二指肠中的酸性食糜 |
| 缩胆囊素（CCK） | 十二指肠到回肠I细胞 | 促进胰酶分泌、胆囊收缩，抑制胃排空 | 小肠中蛋白质和脂肪及其消化产物 |
| 抑胃肽 | 十二指肠和空肠K细胞 | 抑制胃运动和胃液分泌，刺激胰岛素分泌 | 食糜进入十二指肠和空肠 |
| 生长抑素 | 胰岛D细胞 | 对胃肠激素分泌产生广泛抑制作用 | — |
| 胃动素 | 十二指肠和空肠 | 调节摄食后胃肠运动 | 乙酰胆碱 |

### 8. 唾液的生理功能

1. 润湿口腔和饲料，便于咀嚼和吞咽
2. 溶解食物中可溶性成分，刺激味觉产生，引起消化道多种反射活动
3. 唾液淀粉酶在接近中性环境下催化淀粉水解为麦芽糖（入胃后pH未降到4.5之前仍能发挥作用）
4. 幼畜唾液中舌脂酶可水解脂肪为游离脂肪酸
5. 清洁和保护作用（溶菌酶有杀菌作用）
6. 维持口腔碱性环境，保护饲料中的碱性酶
7. 反刍动物大量碱性唾液中和瘤胃发酵产生的酸
8. 某些动物借助唾液蒸发调节体温
9. 反刍动物有大量尿素经唾液进入瘤胃，参与尿素再循环

### 9. 胃液的主要成分及作用

| 成分 | 分泌细胞 | 主要作用 |
| --- | --- | --- |
| 胃蛋白酶原 | 主细胞 | 经盐酸激活变为胃蛋白酶，最适pH1.5～3.5，分解蛋白质为际和陈；对乳中酪蛋白有凝固作用 |
| 盐酸 | 壁细胞 | ①激活胃蛋白酶原；②提供酸性环境，使蛋白质变性；③杀菌；④促进胰液、胆汁、小肠液分泌；⑤促进Fe³⁺还原为Fe²⁺，有助于铁和钙吸收 |
| 黏液与碳酸氢盐 | 表面上皮细胞、颈黏液细胞等 | 形成黏液-碳酸氢盐屏障，防止胃酸和胃蛋白酶侵蚀黏膜 |
| 内因子 | 壁细胞 | 与维生素B₁₂结合，促进其在回肠的吸收 |

### 10. 壁细胞分泌盐酸的机制

1. 壁细胞内H₂O解离→H⁺和OH⁻
2. H⁺由质子泵（H⁺-K⁺-ATP酶）泵出至分泌小管腔
3. 细胞内CO₂ + H₂O →(碳酸酐酶)→ H₂CO₃ → H⁺ + HCO₃⁻
4. HCO₃⁻在细胞底侧与Cl⁻交换进入血液
5. Cl⁻通过细胞顶膜特异Cl⁻通道进入分泌小管腔
6. H⁺ + Cl⁻ → HCl
7. K⁺经顶膜K⁺通道进入分泌小管（质子泵分泌前提）
8. Na⁺-K⁺-ATP酶补充细胞内K⁺

**关键**：质子泵是各种因素引起胃酸分泌的最后通路→奥美拉唑选择性抑制质子泵→抑制胃酸分泌

### 11. 消化期胃液分泌的三期

| 时期 | 刺激部位 | 调节方式 | 特点 |
| --- | --- | --- | --- |
| 头期 | 口腔感受器 | 条件反射+非条件反射，迷走神经传出 | 持续时间长、分泌量大、酸度高、胃蛋白酶含量高、消化力强 |
| 胃期 | 胃底/胃体/幽门部感受器 | 迷走-迷走反射+壁内神经丛反射+胃泌素 | 酸度较高，但含酶量较头期少，消化力较弱 |
| 肠期 | 十二指肠 | 体液调节为主（胃泌素、肠泌酸素、氨基酸） | 分泌量少 |

**头期迷走神经的双重递质**：
- 支配壁细胞：释放乙酰胆碱→直接刺激胃酸分泌（可被阿托品阻断）
- 支配G细胞：释放蛙皮素/胃泌素释放肽（GRP）→促进胃泌素释放（不被阿托品阻断）

### 12. 促进和抑制胃液分泌的因素

**促进因素**：
| 因素 | 作用机制 |
| --- | --- |
| 乙酰胆碱 | 迷走神经末梢释放，直接作用于壁细胞M受体 |
| 胃泌素 | G细胞释放，经血液循环作用于壁细胞，Ca²⁺依赖性途径 |
| 组胺 | 肠嗜铬样细胞分泌，局部扩散至壁细胞，cAMP途径；乙酰胆碱和胃泌素可促进其释放 |

**抑制因素**：
| 因素 | 作用机制 |
| --- | --- |
| 盐酸 | 胃窦pH降至1.2～1.5时：直接抑制G细胞+刺激D细胞释放生长抑素间接抑制；十二指肠pH下降时：刺激胰泌素和球抑胃素 |
| 脂肪 | 进入小肠后刺激抑胃肽等释放，抑制胃液分泌（肠抑胃素） |
| 高渗溶液 | 十二指肠内高渗→刺激渗透压感受器→肠-胃反射抑制分泌 |
| 生长抑素 | 抑制胃泌素和组胺的释放 |

**关键**：盐酸是胃腺分泌的产物，反过来抑制胃腺分泌→负反馈调节→防止胃酸过度分泌

### 13. 胃运动的形式及生理意义

| 运动形式 | 特点 | 生理意义 |
| --- | --- | --- |
| 容受性舒张 | 进食时迷走神经反射引起胃底和胃体舒张 | 容纳大量食物而胃内压不明显升高 |
| 紧张性收缩 | 胃壁平滑肌经常保持微弱持续收缩 | 维持胃内压、胃形状和位置；是其他运动的基础 |
| 蠕动 | 进食后开始，从胃中部向幽门方向推进 | 磨碎食物、与胃液混合、推进食糜向十二指肠 |

### 14. 胃排空及其调节

- **排空速度**：糖类 > 蛋白质 > 脂肪（脂肪排空最慢）
- **促进排空**：胃内食物→壁内神经丛和迷走-迷走反射→胃运动增强
- **抑制排空**：十二指肠内食糜→肠-胃反射+胰泌素、抑胃肽、CCK等→抑制胃运动

### 15. 瘤胃的环境特点

- 营养物质丰富，渗透压与血浆相近
- 温度适宜：38～41℃
- pH维持在6～7（VFA被吸收和碱性唾液缓冲）
- 高度厌氧环境（CO₂、甲烷为主，少量氧很快被微生物利用）
- 可看做供嫌气性微生物高效率繁殖的活体发酵罐

### 16. 瘤胃微生物及其作用

**纤毛虫**：
- 含多种酶：分解糖类、蛋白分解酶、纤维素分解酶
- 能发酵糖、果胶、纤维素和半纤维素，产生乙酸、丙酸、乳酸、CO₂和氢
- 也能降解蛋白质、水解脂类、氢化不饱和脂肪酸
- 纤毛虫体蛋白品质超过菌体蛋白（富含赖氨酸等必需氨基酸）
- 约可为宿主提供动物性蛋白需要量的20%
- pH降至5或更低时，纤毛虫活力降低、数量减少或完全消失
- 幼畜通过与亲畜接触自然感染获得纤毛虫

**细菌**：
- 数量最多（1g瘤胃内容物含10¹⁰～10¹¹个）
- 发酵糖类：可溶性糖最快→淀粉次之→纤维素和半纤维素最慢
- 利用非蛋白含氮物（尿素、铵盐等）合成微生物蛋白质
- 微生物蛋白质富含多种必需氨基酸
- 成年牛一昼夜进入皱胃的微生物蛋白质约100g

**微生物间的共生关系**：
- 纤毛虫能吞噬和消化细菌
- 细菌之间也存在共生（如白色瘤胃球菌消化纤维素产己糖→满足反刍兽拟杆菌能量需要；反刍兽拟杆菌消化蛋白质→为白色瘤胃球菌提供氨基酸和氨）
- 宿主与微生物之间也存在共生关系

### 17. 瘤胃内的消化代谢过程

**（1）糖类的分解和利用**：
- 纤维素→纤维二糖→葡萄糖→丙酮酸/乳酸→VFA + CH₄ + CO₂
- VFA主要是乙酸、丙酸、丁酸
- 通常乙酸:丙酸:丁酸 = 70:20:10
- 粗饲料多时乙酸/丙酸比例升高；蛋白质饲料多时乙酸比例下降、丁酸上升
- VFA是反刍动物主要能源物质，占所需能量的60%～70%
- VFA被瘤胃壁吸收进入血液循环
- 微生物还能将单糖和双糖转化成自身糖原→进入皱胃和小肠后被消化吸收

**（2）蛋白质的分解与微生物蛋白质的合成**：
- 50%～70%饲料蛋白质被微生物蛋白酶水解为肽和氨基酸
- 大部分氨基酸脱氨基→NH₃ + CO₂ + 短链脂肪酸
- NH₃是合成微生物蛋白质的主要氮源
- 合成蛋白质还需能量和碳链（VFA、CO₂、糖）
- 尿素再循环：NH₃→门脉→肝脏→尿素→唾液/瘤胃壁→瘤胃→微生物利用
- 添加尿素注意事项：脲酶分解尿素速度约为微生物利用速度的4倍→不宜过多→防氨中毒

**（3）脂类的消化**：
- 饲料中甘油三酯被微生物水解为甘油和脂肪酸
- 甘油发酵生成丙酸
- 不饱和脂肪酸被氢化→转变成饱和脂肪酸
- 反刍动物体脂中饱和脂肪酸比例高达55%～62%

**（4）维生素的合成**：
- 瘤胃微生物能合成多种B族维生素和维生素K
- 幼龄反刍动物瘤胃发育不完善，可能患B族维生素缺乏症

**（5）气体的产生**：
- 牛一昼夜产气600～1300L
- CO₂占50%～70%，甲烷占30%～40%
- CO₂来源：糖类发酵、氨基酸脱羧、唾液碳酸氢盐中和脂肪酸
- 甲烷来源：4H₂ + CO₂ → CH₄ + 2H₂O（甲烷细菌作用）

### 18. 前胃运动

**网瘤胃运动**：
- 从网胃两相收缩开始
- 第一相收缩力量较弱→将粗糙饲料压向瘤胃
- 第二相收缩十分强烈→内腔几乎消失（异物易造成创伤性网胃炎和心包炎）
- 反刍时在两相收缩前出现额外附加收缩
- 瘤胃A波：从瘤胃前部→后背囊→后腹囊→腹囊由后向前→前部（食物混合移动）
- 瘤胃B波：起于后部，常伴嗳气

**瘤胃内容物分层**：气体→固体层（漂浮状）→悬浮层→液体层

**瓣胃运动**：与网瘤胃协调，瓣胃沟收缩与瘤胃背囊收缩同步，瓣胃和叶片收缩起研磨作用，具有强大吸收功能（吸收残存VFA和碳酸氢盐）

**前胃运动调节**：基本中枢在延髓，高级中枢在大脑皮层；迷走神经加强运动，交感神经抑制运动

### 19. 反刍过程

1. **逆呕**：网胃额外收缩→食团上升到贲门→贲门扩张→伴随吸气胸内负压加大→食管内压下降→食团被驱入食管→食管逆蠕动→食团以约1m/s速度返回口腔
2. **再咀嚼**：食团在口腔中仔细咀嚼
3. **再混唾液**：咀嚼时混入大量唾液
4. **再吞咽**：食团重新吞入瘤胃

**反刍的生理意义**：将粗糙饲料充分磨碎，增加微生物作用面积；混入大量碱性唾液中和瘤胃发酵产生的酸

### 20. 胰液的主要成分及作用

| 成分 | 作用 |
| --- | --- |
| 碳酸氢盐 | 中和进入十二指肠的胃酸，为小肠内消化酶提供适宜碱性环境 |
| 胰蛋白酶原+糜蛋白酶原 | 被肠致活酶激活后，分解蛋白质为小肽和氨基酸 |
| 羧肽酶 | 分解多肽为氨基酸 |
| 胰脂肪酶 | 分解甘油三酯为脂肪酸、甘油和甘油一酯 |
| α-淀粉酶 | 分解淀粉为麦芽糖 |
| 核糖核酸酶/脱氧核糖核酸酶 | 分解RNA/DNA为单核苷酸 |
| 胆固醇酯酶 | 分解胆固醇酯为胆固醇和脂肪酸 |
| 磷脂酶 | 分解磷脂为脂肪酸和溶血磷脂 |

**关键**：胰液含有消化三大营养物质的所有酶→是最重要的消化液

### 21. 胰液分泌的调节

**神经调节**：迷走神经兴奋→乙酰胆碱→促进胰液分泌（含酶丰富但水量少）

**体液调节**：
| 激素 | 作用 |
| --- | --- |
| 胰泌素 | 酸性食糜刺激S细胞→促进胰腺分泌大量碳酸氢盐（水多酶少） |
| 缩胆囊素（CCK） | 蛋白质和脂肪消化产物刺激I细胞→促进胰酶分泌（水少酶多） |

### 22. 胆汁的生理作用

胆汁中不含消化酶，主要通过胆盐发挥作用：
1. **乳化剂**：降低脂肪表面张力→脂肪裂解为3～10μm微滴→增加与胰脂肪酶接触面积
2. **形成微胶粒**：脂肪酸、甘油一酯掺入微胶粒→形成水溶性复合物→便于吸收
3. **增强脂肪酶活性**：脂肪酶激活剂
4. **促进脂溶性维生素（A、D、E、K）吸收**
5. **刺激小肠运动**

**胆盐的肠-肝循环**：胆盐排出→小肠→90%以上被吸收→门静脉→肝脏→重新组成胆汁→排入十二指肠。每循环一次5%～10%随粪便排出。返回的胆盐能刺激胆汁分泌。

### 23. 小肠运动的形式

| 运动形式 | 发生时期 | 特点 | 生理意义 |
| --- | --- | --- | --- |
| 紧张性收缩 | 消化期 | 平滑肌经常处于紧张状态 | 小肠运动的基础 |
| 分节运动 | 消化期 | 环行肌多点同时收缩，食糜不断分开混合 | 使食糜与消化液混合；与肠壁紧密接触利于吸收；挤压肠壁助血液和淋巴回流 |
| 蠕动 | 消化期 | 缓慢波浪式推进，推进距离短 | 推进食糜到新肠段 |
| 蠕动冲 | 消化期 | 速度快、传播远 | 将内容物从起始部推送到末端 |
| 逆蠕动 | 消化期 | 与蠕动方向相反 | 使食糜来回移动，充分消化吸收 |
| MMC | 消化间期 | 强有力蠕动性收缩，传播远 | 推送未消化物质离开小肠；控制前段肠管细菌数量 |

**分节运动频率梯度**：前段频率高，后段频率低，与慢波电位从十二指肠到回肠末端逐渐降低有关

### 24. 小肠运动的调节

| 调节方式 | 机制 |
| --- | --- |
| 内在神经丛 | 主要调节作用（尤其是肌间神经丛）；兴奋性神经元（ACh、P物质）+ 抑制性神经元（VIP、NO） |
| 外来神经 | 迷走神经促进运动；交感神经抑制运动（但与肠肌当时状态有关） |
| 体液因素 | 促进：ACh、5-HT、胃泌素、CCK、胃动素、P物质；抑制：VIP、抑胃肽、内啡肽、胰泌素、肾上腺素、胰高血糖素 |

### 25. 大肠内微生物消化的意义

- 对草食动物（尤其是马属动物和兔）非常重要
- 马的大肠：40%～50%纤维素、39%蛋白质、24%糖被消化
- 大肠微生物消化纤维效率达反刍动物的60%～70%
- 马需要的能量至少一半由盲肠和结肠微生物发酵供应
- 大肠微生物也能合成B族维生素和维生素K
- 存在类似瘤胃的尿素再循环方式

### 26. 小肠作为主要吸收部位的有利条件

1. **面积大**：皱褶、绒毛、微绒毛→总吸收面积巨大
2. **血液和淋巴丰富**：绒毛内有丰富的毛细血管和中央乳糜管
3. **食物停留时间长**：食物在小肠内停留3～8h
4. **食物已被充分消化**：三大营养物质已被分解为可吸收的小分子
5. **小肠绒毛的泵吸作用**：平滑肌纤维收缩和舒张→绒毛伸缩→泵吸作用促进吸收

### 27. 三大营养物质的吸收

**（1）糖的吸收**：
- 只有单糖（葡萄糖、半乳糖、果糖）能被吸收
- 葡萄糖和半乳糖：与Na⁺耦联的继发性主动转运（Na⁺-葡萄糖同向转运体）
- 果糖：通过载体易化扩散，不与Na⁺耦联
- 吸收速率：半乳糖 > 葡萄糖 > 果糖

**（2）蛋白质的吸收**：
- 游离氨基酸：主动吸收，与Na⁺吸收耦联的继发性主动转运，4种特殊载体
- 二肽和三肽：也可完整被吸收，速率比氨基酸快，耗能低、容量大、不易饱和
- 新生动物：可通过胞饮作用完整吸收初乳中蛋白质（免疫球蛋白），获得被动免疫，24～36h后此能力消失
- 吡哆醛（维生素B₆）参与氨基酸主动转运

**（3）脂类的吸收**：
- 脂肪消化产物与胆盐形成混合微胶粒→通过静水层→靠近上皮细胞
- 脂肪酸、甘油一酯、胆固醇以扩散方式进入上皮细胞
- 中短链脂肪酸（<10～12C）→直接进入血液循环
- 长链脂肪酸和甘油一酯→在内质网重新合成甘油三酯→与载脂蛋白形成乳糜微粒→进入中央乳糜管→淋巴循环
- **脂肪吸收以淋巴途径为主**

### 28. 无机盐和水分的吸收

**钠的吸收**（3种机制）：
1. Na⁺耦联转运系统（与葡萄糖、氨基酸等耦联的主动转运）
2. Na⁺-Cl⁻同时吸收（H⁺-Na⁺交换 + HCO₃⁻-Cl⁻交换）
3. Na⁺简单扩散（借电化学梯度）

**钙的吸收**：
- 主动转运，速度比钠慢
- 钙通道进入上皮细胞→钙泵泵出至血液
- 钙结合蛋白（Ca-BP）参与转运
- 溶解状态下才能吸收，pH≈3时吸收最好
- 胃酸促进钙吸收；磷酸盐过多形成不溶性磷酸钙→妨碍吸收

**铁的吸收**：
- 主要在十二指肠和空肠
- 亚铁（Fe²⁺）吸收速度比高铁（Fe³⁺）快2～5倍
- 维生素C将Fe³⁺还原为Fe²⁺促进吸收
- 胃酸促进铁吸收
- 转铁蛋白参与转运；铁蛋白防止铁过量吸收

**水分的吸收**：被动吸收，动力为溶质（特别是NaCl）主动吸收产生的渗透压梯度

## 🔄 三、流程类考点

**壁细胞分泌盐酸完整流程：**
1. 壁细胞内H₂O解离→H⁺ + OH⁻
2. H⁺由质子泵（H⁺-K⁺-ATP酶）泵出至分泌小管腔
3. CO₂ + H₂O →(碳酸酐酶)→ H₂CO₃ → H⁺ + HCO₃⁻
4. HCO₃⁻在细胞底侧与Cl⁻交换→进入血液
5. Cl⁻通过顶膜特异Cl⁻通道进入分泌小管腔
6. K⁺经顶膜K⁺通道进入分泌小管（质子泵前提）
7. Na⁺-K⁺-ATP酶补充细胞内K⁺
8. H⁺ + Cl⁻ → HCl

---

**消化期胃液分泌调节流程：**
1. **头期**：食物刺激口腔感受器→条件反射+非条件反射→迷走神经→①ACh→壁细胞→胃酸分泌；②GRP→G细胞→胃泌素→壁细胞→胃酸分泌
2. **胃期**：食物入胃→①迷走-迷走反射+壁内神经丛反射→胃液分泌；②扩张和化学刺激→G细胞→胃泌素→胃液分泌
3. **肠期**：食糜入十二指肠→G细胞分泌少量胃泌素+肠泌酸素+吸收的氨基酸→胃液分泌
4. **反馈抑制**：胃内pH降至1.2～1.5→直接抑制G细胞+刺激D细胞→生长抑素→抑制胃泌素→胃酸分泌减少

---

**瘤胃内糖类消化代谢流程：**
1. 饲料中纤维素、淀粉、糖类进入瘤胃
2. 微生物发酵：纤维素→纤维二糖→葡萄糖→丙酮酸/乳酸
3. 最终产物：VFA（乙酸、丙酸、丁酸）+ CH₄ + CO₂
4. VFA被瘤胃壁吸收→进入血液循环→供能
5. 微生物将单糖/双糖转化为自身糖原→进入皱胃和小肠→被消化吸收
6. 乙酸:丙酸:丁酸 ≈ 70:20:10

---

**尿素再循环完整流程：**
1. 瘤胃内蛋白质被微生物水解→肽和氨基酸→脱氨基→NH₃
2. NH₃被瘤胃壁吸收→经门脉循环→进入肝脏
3. 肝脏通过鸟氨酸循环→NH₃ + CO₂ → 尿素
4. 尿素经血液循环→①唾液腺→随唾液进入瘤胃；②通过瘤胃壁弥散进入瘤胃
5. 瘤胃内尿素被微生物脲酶分解→NH₃
6. NH₃被微生物利用→合成微生物蛋白质
7. 微生物蛋白质随食糜进入皱胃和小肠→被宿主消化吸收

---

**胆盐的肠-肝循环流程：**
1. 肝细胞分泌胆汁→胆盐排入十二指肠
2. 胆盐在小肠内发挥消化作用（乳化脂肪、形成微胶粒等）
3. 90%以上胆盐由小肠黏膜吸收入血
4. 经门静脉回到肝脏
5. 肝脏重新组成胆汁→排入十二指肠
6. 每循环一次5%～10%随粪便排出
7. 返回的胆盐刺激胆汁分泌

---

**脂肪在小肠内消化吸收流程：**
1. 脂肪进入小肠→胆盐乳化→脂肪裂解为3～10μm微滴
2. 胰脂肪酶水解→脂肪酸 + 甘油一酯
3. 脂肪消化产物与胆盐形成混合微胶粒
4. 微胶粒通过静水层→靠近上皮细胞
5. 脂肪酸、甘油一酯、胆固醇以扩散方式进入上皮细胞
6. 中短链脂肪酸→直接进入血液循环
7. 长链脂肪酸+甘油一酯→内质网重新合成甘油三酯→与载脂蛋白形成乳糜微粒
8. 乳糜微粒以出胞方式→中央乳糜管→淋巴循环

## 🧬 四、 核心名词解释（严格原文）

* **【消化】**：食物中的营养物质在消化道内被分解为可吸收的小分子物质的过程称为消化。
* **【吸收】**：消化后的产物透过消化道黏膜上皮进入血液和淋巴循环的过程称为吸收。
* **【机械性消化】**：机械性消化也称物理性消化，是指通过咀嚼和消化道肌肉的舒缩活动来完成的消化活动，其主要作用是将食物磨碎，并使之与消化液充分混合，为化学性消化创造条件；同时使消化道的内容物不断地向消化道远端移送。
* **【化学性消化】**：化学性消化是指通过消化腺所分泌的消化液来完成的消化活动。
* **【微生物消化】**：微生物消化也称生物学消化，是指由栖居在动物消化道内的微生物来完成的消化活动。
* **【慢波】**：在安静状态下，用微电极可在消化道平滑肌静息电位的基础上记录到一种缓慢的、大小不等的节律性去极化波，称为慢波或基本电节律（BER），其波幅为10～30mV，持续1～4s。
* **【黏液-碳酸氢盐屏障】**：不溶性黏液具有较高的黏滞性和形成凝胶的特征，内衬于胃腔表面成为厚约1mm的黏液层，与胃黏膜上皮细胞分泌的HCO₃⁻一起构成"黏液-碳酸氢盐屏障"，有效防止了胃酸和胃蛋白酶对黏膜的侵蚀。
* **【内因子】**：内因子是壁细胞分泌的一种糖蛋白，它有两个活性部位，一个与进入胃内的维生素B₁₂结合成复合体，使维生素B₁₂在运送到回肠途中不被消化液中的水解酶所破坏，到达回肠后，内因子另一个活性部位与回肠黏膜刷状缘的特殊受体结合，促进维生素B₁₂吸收入血。
* **【容受性舒张】**：当咀嚼和吞咽食物时，食物对咽、食管等处感受器的刺激，可通过迷走神经反射性地引起胃底和胃体部平滑肌的舒张，称为容受性舒张。
* **【胃排空】**：胃内食糜由胃排入十二指肠的过程称为胃排空。
* **【呕吐】**：呕吐是将胃及肠内容物从口腔强力驱出的动作，是一种具有保护意义的防御反射。
* **【反刍】**：反刍动物在采食时，饲料一般不经充分咀嚼就匆匆吞进瘤胃，饲料在瘤胃经浸泡软化和短时间的发酵，当动物休息时再将食团返回口腔仔细咀嚼，这种特殊的消化活动称为反刍。反刍包括逆呕、再咀嚼、再混唾液和再吞咽4个阶段。
* **【嗳气】**：瘤胃内发酵产生的气体，经食管逆行排出口腔的过程，称为嗳气。
* **【挥发性脂肪酸（VFA）】**：瘤胃微生物发酵糖类产生的有机酸，主要是乙酸、丙酸和丁酸，是反刍动物主要的能源物质。
* **【尿素再循环】**：瘤胃中的氨被瘤胃壁吸收，经门脉循环进入肝脏，通过鸟氨酸循环生成尿素。一部分尿素经唾液和瘤胃壁重新进入瘤胃，被微生物利用，这种内源性的尿素再循环对于提高饲料中含氮物质的利用率具有重要意义。
* **【胆盐的肠-肝循环】**：胆汁中的胆盐排出至小肠后，绝大部分（90%以上）可由小肠黏膜吸收入血，通过门静脉回到肝脏，重新组成胆汁排入十二指肠，这一过程称为胆盐的肠-肝循环。
* **【分节运动】**：分节运动是由肠壁环行肌的收缩和舒张所形成的一种运动方式，使食糜不断地分开又不断地混合，主要作用是使食糜与消化液充分混合，便于化学性消化，并使食糜与肠壁紧密接触，有利于吸收。
* **【蠕动冲】**：在小肠还可见到一种进行速度很快、传播较远的蠕动，称为蠕动冲，它可将小肠内容物从小肠的起始部位一直推送到末端。
* **【移行运动复合波（MMC）】**：发生在消化间期的一种强有力的蠕动性收缩，传播很远，有时能传播至整个小肠。起始于十二指肠，以慢波的频率向后传播，作用是推送未消化的物质离开小肠，控制前段肠管内的细菌数量。
* **【腔期消化】**：由肠腺分泌的肠激活酶和淀粉酶在肠腔内发挥作用，与食糜充分混合后在消化管内发挥作用的消化方式，称为腔期消化。
* **【膜期消化】**：肠黏膜上皮分泌的肠肽酶、二糖酶等，以化学键与上皮细胞顶点膜相连，当腔期消化产物接触小肠黏膜时，使其进一步水解为小分子物质而被吸收，这种在肠上皮细胞表面进行的消化过程称为膜期消化，是腔期消化的继续和补充。
* **【脑肠肽】**：一些胃肠激素如缩胆囊素、P物质等不仅存在于胃肠道内，还存在于脑内；而原先认为只存在于中枢神经系统的神经肽，如生长抑素等，也在消化道发现，这些双重分布的肽被称为脑肠肽。
* **【胃肠激素】**：调节消化道功能的体液因素主要是胃肠激素，由胃至结肠的黏膜层中含有40多种不同类型的内分泌细胞分泌，消化道被认为是体内最大、最复杂的内分泌器官。
* **【迷走-迷走反射】**：胃肠交感神经中约有50%的纤维为传入纤维，迷走神经中约有75%的纤维为传入纤维，可及时将胃肠感受器信号传入高位中枢，引起反射调节，称为"迷走-迷走"反射。
