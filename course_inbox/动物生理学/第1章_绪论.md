# 第一章 绪论

## 🔑 一、核心机制与关键考点

### 1. 动物生理学研究的三个水平
- **核心原理**：生理学研究可在细胞和分子、器官和系统、整体三个水平上进行，三者互相联系、互相补充。
- **机制/步骤详解**：
    1. **细胞和分子水平**：研究细胞及构成细胞的分子的生理特性。如肌细胞收缩是由于某些离子浓度改变及酶的作用下蛋白质分子排列方式变化的结果。多数需离体培养，注意不能简单推论完整机体中的功能。
    2. **器官和系统水平**：研究器官或系统的功能、在机体中的作用、功能活动的内在机制及影响因素。如循环生理学、消化生理学、肾脏生理学等。
    3. **整体水平**：以完整机体为研究对象，观察和分析在各种环境条件下不同器官、系统之间互相联系、互相协调，以及完整机体对环境变化发生各种反应的规律。
- **关键注意点**：
    - 三个水平的研究不是相互孤立的，而是互相联系、互相补充的
    - 可以用基因转移和基因敲除的方法建立转基因动物和基因敲除动物，在整体中观察和研究各种基因的功能

### 2. 急性实验与慢性实验的比较
| 对比维度 | 急性实验 | 慢性实验 |
| --- | --- | --- |
| 分类 | 离体器官实验（in vitro）、活体解剖实验（in vivo） | 无菌条件下手术，长期观察 |
| 持续时间 | 不能持久，实验后动物死亡 | 可在较长时间内用于实验 |
| 优点 | 实验条件简单，可尽量消除与研究无关的因素 | 更接近正常生理状态，可在动物清醒条件下长期观察 |
| 不足 | 不能完全代表正常生理条件下的功能状态，属于分析性研究 | 不便于分析诸多影响因素 |

### 3. 生命现象的四种基本特征
- **核心原理**：生命现象至少包括4种基本活动：新陈代谢、兴奋性、生殖和适应。
- **机制/步骤详解**：
    1. **新陈代谢**：是最显著的生命特征。包括物质代谢和能量代谢，二者始终伴随。物质分解时释放能量，物质合成时吸收能量。新陈代谢逐步减弱→衰老；停止→死亡。
    2. **兴奋性**：机体或组织对刺激发生反应的能力。近代生理学中，兴奋性被理解为细胞在受刺激时产生动作电位的能力，兴奋一词成为产生动作电位的过程或动作电位的同义词。
    3. **生殖**：产生与自己相似的子代个体。单细胞生物通过分裂或有丝分裂；高等动物由两性生殖细胞结合。
    4. **适应**：根据外界情况调整内部关系的生理特性，目的是保持内环境的稳定。

### 4. 兴奋性概念的演变
- **核心原理**：兴奋性概念从"对刺激发生反应的能力"发展为"受刺激时产生动作电位的能力"。
- **机制/步骤详解**：
    1. **经典定义**：活组织或细胞对外界刺激发生反应的能力。
    2. **现代定义**：细胞在受刺激时产生动作电位的能力。
    3. **兴奋一词的新含义**：产生动作电位的过程或动作电位的同义词。
    4. **可兴奋组织的判断标准**：只有那些在受刺激时能出现动作电位的组织，才能称为可兴奋组织；只有组织产生了动作电位时，才能说组织产生了兴奋。

### 5. 刺激引起兴奋的三个参数及其相互关系
- **核心原理**：刺激要引起组织细胞产生兴奋，必须要求三个参数达到某一临界值：刺激的强度、刺激的持续时间、刺激强度对于时间的变化率。
- **机制/步骤详解**：
    1. 在强度-时间变化率保持不变的情况下，引起组织兴奋所需的最小刺激强度与刺激持续时间呈反比关系。
    2. 刺激强度减小到某一数值时，不论持续多长时间也不会引起组织兴奋（基强度）。
    3. 刺激持续时间缩短到某一临界值时，无论使用多么大的刺激强度，也不能引起组织兴奋。
    4. 用方波刺激时，强度-时间变化率固定不变，只需考虑强度和持续时间两个参数。
- **关键影响因素**：
    - 阈值是衡量组织兴奋性高低的指标
    - 组织的兴奋性与阈值呈反比关系
    - 阈值越小→兴奋性越高；阈值越大→兴奋性越低

### 6. 内环境与稳态
- **核心原理**：细胞外液是细胞直接生活的环境（内环境），其理化性质及各种物质的浓度保持在相对稳定的状态（稳态）。
- **机制/步骤详解**：
    1. 内环境稳态是一种动态的相对稳定。
    2. 理论发展：1852年贝尔纳推测内环境恒定主要依赖神经调节→1929年坎农提出神经调节和体液调节共同作用→后来发现免疫系统也参与调节。
    3. **神经-体液-免疫调节网络**是内环境稳态的主要调节机制。
    4. 直接相关的系统：消化系统、呼吸系统、循环系统、泌尿系统。
    5. 起调节作用的系统：神经系统、内分泌系统、免疫系统。
- **关键注意点**：
    - 内环境稳态失调的病例：尿毒症、高山缺氧反应、发高烧、严重腹泻、酸碱中毒
    - 血糖过低→低血糖症；血糖过高→糖尿病
    - 血钙过低→影响骨发育，出现佝偻病；血钙过高→肌无力
    - 肾衰竭→尿毒症→死亡

### 7. 神经调节的特点与机制
- **核心原理**：神经调节的基本过程是反射，反射的结构基础是反射弧。
- **机制/步骤详解**：
    1. 感受器感受体内某部位或外界环境的变化，转变成神经信号（电活动）
    2. 通过传入神经传导至相应的神经中枢
    3. 神经中枢对传入信号进行分析，做出反应
    4. 通过传出神经改变效应器（如肌肉、腺体）的活动
    5. 反射弧任何一部分被破坏，都会导致反射活动消失
- **神经调节的特点**：反应迅速、反应准确、作用部位局限、作用时间短暂

### 8. 体液调节与神经-体液调节
- **核心原理**：体液调节通过激素等化学物质经体液运输作用于靶细胞，与神经调节密切联系形成神经-体液调节。
- **机制/步骤详解**：
    1. **内分泌腺细胞分泌激素**→血液或组织液携带→作用于靶细胞上相应受体→调节细胞活动
    2. **旁分泌调节**：激素在组织液中扩散至邻近细胞，调节邻近细胞活动（局部性体液调节）
    3. **神经分泌**：下丘脑神经内分泌神经元合成分泌激素，经轴突末梢释放入血液
    4. **神经-体液调节**：神经系统与内分泌系统功能上关系密切，有相互调节作用
- **关键考点——三级管理功能轴**：
    - 下丘脑-腺垂体-甲状腺功能轴
    - 下丘脑-腺垂体-肾上腺皮质功能轴
    - 下丘脑-腺垂体-性腺功能轴
- **关键注意点**：
    - 下丘脑某些神经内分泌神经元受神经递质调节：多巴胺、乙酰胆碱、5-羟色胺、β-内啡肽、强啡肽
    - 肾上腺髓质受交感节前纤维支配
    - 甲状腺腺泡、肾近球小体颗粒细胞受交感神经纤维支配
    - 胃泌素G细胞、胰岛B细胞和A细胞受迷走神经和交感神经双重支配
    - 激素对神经调节的允许作用：交感神经末梢释放去甲肾上腺素使血管收缩的作用，需要有肾上腺糖皮质激素存在

### 9. 自身调节的实例
- **核心原理**：组织、细胞本身对周围环境变化发生适应性反应，不依赖于外来的神经或体液因素。
- **实例**：
    1. **血管平滑肌自身调节**：小动脉灌注压力升高→血管壁牵张刺激增强→血管平滑肌收缩→口径缩小→血流量不致增大
    2. **肾脏小动脉自身调节**：动脉血压在一定范围内变动时，肾血流量能保持相对稳定
    3. **甲状腺自身调节**：血浆中碘浓度改变时，甲状腺自身调节对碘的摄取以及合成和释放甲状腺激素的能力

### 10. 三种控制系统的比较
| 对比维度 | 非自动控制系统 | 反馈控制系统 | 前馈控制系统 |
| --- | --- | --- | --- |
| 性质 | 开环系统（open-loop system） | 闭环系统 | 快捷途径 |
| 特点 | 受控部分活动不会反过来影响控制部分，单方向 | 受控部分的活动作为反馈信号送回控制部分 | 干扰信息同时通过快捷途径作用于受控部分 |
| 分类 | 无 | 负反馈、正反馈 | 无 |
| 生理意义 | 在正常生理功能调节中不多见 | 维持内环境稳态（负反馈）或破坏平衡（正反馈） | 使调控过程不出现大的波动和反应滞后 |

### 11. 负反馈控制系统
- **核心原理**：负反馈使受控部分的活动向原先活动相反的方向改变，维持系统的活动稳定。
- **机制/步骤详解**：
    1. 受控部分活动增强→感受装置反馈信息→控制部分发出指令使受控部分活动减弱→向原先平衡状态恢复
    2. 受控部分活动过低→负反馈机制使其活动增强→向原先平衡状态恢复
- **关键实例**：
    - **动脉血压的负反馈调节**：血压升高→动脉压力感受器传入冲动增加→心血管中枢活动改变→心脏活动减弱、血管舒张→血压恢复
    - **体位变化时的血压调节**：起卧转位→血液滞留下肢静脉→回心血量减少→血压降低→压力感受器传入冲动减少→心脏活动加强、血管收缩→血压回升
- **调定点与重调定**：
    - 负反馈机制以调定点为参照水平
    - 原发性高血压：血压调定点被设置在较高水平→血压保持高于正常
    - 调定点发生变动的过程称为重调定

### 12. 正反馈控制系统
- **核心原理**：正反馈使受控部分继续加强向原来方向的活动，破坏原先的平衡状态。
- **正常生理实例**：
    1. **血液凝固**：血管破裂→各种凝血因子相继激活→形成血凝块封住破口
    2. **分娩过程**：子宫收缩→胎儿头部下降牵张子宫颈→子宫颈牵张进一步加强子宫收缩→反复直至胎儿娩出
- **病理情况下的正反馈（恶性循环）**：
    - 大量失血→心脏射血量减少→血压降低→冠状动脉血流量减少→心肌收缩力减弱→心脏射血量更少→反复→死亡

### 13. 前馈控制系统
- **核心原理**：干扰信息在引起输出效应变化的同时，通过另一快捷途径作用于受控部分，使调整发生在反馈信息之前。
- **关键实例**：
    - 动物见到食物引起唾液分泌（比食物入口引起的唾液分泌更快，具有预见性）
- **生理意义**：使机体的调控过程不致出现大的波动和反应滞后现象，更有效地保持生理功能的相对稳定

## 📊 二、对比类考点

### 三种调节方式的比较
| 对比维度 | 神经调节 | 体液调节 | 自身调节 |
| --- | --- | --- | --- |
| 作用方式 | 反射（反射弧） | 激素等化学物质经体液运输 | 组织细胞本身的生理特性 |
| 反应速度 | 迅速 | 较慢 | 较慢 |
| 作用范围 | 准确、局限 | 较广泛 | 局限 |
| 作用时间 | 短暂 | 持久 | 较短 |
| 依赖因素 | 依赖神经系统和反射弧完整性 | 依赖激素和受体 | 不依赖外来神经或体液因素 |

### 负反馈与正反馈的比较
| 对比维度 | 负反馈 | 正反馈 |
| --- | --- | --- |
| 调节方向 | 向原先活动相反方向改变 | 继续加强原来方向的活动 |
| 生理意义 | 维持系统活动稳定，维持内环境稳态 | 破坏原先平衡状态 |
| 体内数量 | 绝大多数控制系统 | 仅有很少几个 |
| 典型实例 | 动脉血压调节、体温调节 | 血液凝固、分娩过程 |
| 病理意义 | 稳态失调时功能紊乱 | 恶性循环（如大量失血→心衰→死亡） |

## 🧬 四、核心名词解释（严格原文）

* **【动物生理学】**：通过研究机体内部各系统的生命活动及其相互联系以及动物与外界环境的关系来认识动物机体的正常生命活动规律，从而利用这些规律提高动物的生产性能，更好地为人类的生产活动服务。
* **【新陈代谢】**：机体与周围环境之间不断地进行物质和能量交换，以实现自我更新的过程。它包括合成代谢（同化作用）和分解代谢（异化作用）两个方面。
* **【合成代谢（同化作用）】**：机体不断从外环境中摄取营养物质来合成自身成分的过程。
* **【分解代谢（异化作用）】**：机体不断分解自身物质，并将废物排出体外的过程。
* **【兴奋性】**：机体或组织对刺激发生反应的能力或特性。
* **【兴奋】**：机体或组织接受刺激后，功能活动由弱变强或由静止到活动的变化。
* **【抑制】**：组织受刺激后，组织或机体活动减弱或变为相对静止。
* **【刺激】**：能引起组织细胞发生反应的各种内外环境的变化。
* **【阈值（阈强度/阈刺激）】**：引起组织兴奋，即产生动作电位所需的最小刺激强度。阈值越小，说明该组织越易兴奋，即兴奋性越高；反之，阈值越大，说明组织兴奋性越低。组织的兴奋性与阈值呈反比关系。
* **【阈下刺激】**：强度小于阈值的刺激。阈下刺激不能引起兴奋或动作电位，但并非对组织细胞不产生任何影响。
* **【可兴奋细胞/可兴奋组织】**：只需接受较小程度的刺激，就能表现出某种形式的反应的神经和肌细胞，以及某些腺细胞。
* **【动作电位】**：各种可兴奋细胞处于兴奋状态时，受刺激处的细胞膜两侧出现的一个特殊形式的电变化。动作电位是大多数可兴奋细胞受刺激时共有的特征性表现，它不是细胞其他功能变化的伴随物，而是细胞表现其他功能的前提或触发因素。
* **【生殖（自我复制）】**：机体在生长发育到一定阶段后能够产生与自己相似的子代个体的功能。
* **【适应】**：生物体能够根据外界情况调整其内部关系的生理特性。动物机体针对外环境的变化调整体内自身各部分生理功能和活动的过程。
* **【内环境】**：细胞外液是细胞直接生活的环境，称为内环境。
* **【稳态】**：内环境的理化性质及各种物质的浓度保持在相对稳定的状态。
* **【神经调节】**：通过神经系统的活动对机体各组织、器官和系统的生理功能所发挥的调节作用。
* **【反射】**：在中枢神经系统的参与下，机体对内外环境变化产生的有规律的适应性反应。
* **【反射弧】**：反射的结构基础，包括感受器、传入神经、神经中枢、传出神经和效应器5个基本环节。
* **【非条件反射】**：先天遗传的同类动物都具有的反射活动，是一种初级的神经活动。
* **【条件反射】**：后天获得的，是大脑的高级的神经活动。
* **【体液调节】**：体内的一些细胞能合成并分泌某些特殊的化学物质，由体液运输到达全身的组织细胞或某些特殊的组织细胞，通过作用于细胞上相应的受体，对这些细胞的活动进行调节。
* **【激素】**：一些能在细胞与细胞之间传递信息的化学物质，由血液或组织液携带，作用于具有相应受体的细胞，调节这些细胞的活动。
* **【靶细胞】**：接受某种激素调节的细胞，称为该种激素的靶细胞。
* **【旁分泌调节】**：有一些激素可以在组织液中扩散至邻近的细胞，调节邻近细胞的活动。这种调节是局部性的体液调节。
* **【神经分泌】**：下丘脑某些神经内分泌神经元合成和分泌激素，并将其由轴突末梢释放入血液的分泌方式。
* **【自身调节】**：许多组织、细胞自身也能对周围环境变化发生适应性的反应，这种反应是组织、细胞本身的生理特性，并不依赖于外来的神经或体液因素的作用。
* **【负反馈调节】**：经过反馈调节，受控部分的活动向和它原先活动相反的方向发生改变的调节方式。负反馈控制系统的作用是使系统的活动保持稳定。
* **【正反馈调节】**：反馈调节使受控部分继续加强向原来方向的活动的调节方式。正反馈控制不是维持系统的稳态或平衡，而是破坏原先的平衡状态。
* **【调定点】**：体内许多负反馈调节机制中设置的一个参照水平，即规定受控部分的活动只能在靠近调定点的一个狭小范围内变动。
* **【重调定】**：生理学中将调定点发生变动的过程称为重调定。
* **【前馈控制】**：在某一方面的信息（干扰信息）作用于受控部分引起输出效应发生变化的同时，又通过另一快捷途径作用于受控部分，使其及时地调整活动。
* **【离体器官实验（in vitro）】**：从活的动物体内取出器官、组织或细胞，置于与体内环境相似的人工模拟环境中，使其在短时间内保持生理功能，以便进行研究。
* **【活体解剖实验（in vivo）】**：在麻醉或毁坏大脑的情况下，暴露所要研究的器官，以便进行各种实验。
* **【慢性实验】**：在无菌条件下对健康动物进行手术，并在不损害动物机体完整性的前提下暴露、摘除、破坏以及移植所要研究的器官，然后在可能接近正常的生活条件下，观察实验动物的功能或功能紊乱等。
